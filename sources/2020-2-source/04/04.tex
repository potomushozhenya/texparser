{\leftskip=7mm\noindentРабота посвящена математическому
моделированию двумерной периодической системы полевых эмиттеров.
Электронно-вакуумные приборы на основе полевой эмиссии активно
используются во многих областях науки и~техники. В~данной статье
исследуется полевой катод, представляющий собой периодическую
систему эмиттеров бесконечной длины и~одинаковой формы,
расположенных на плоской подложке на одном и~том же расстоянии
друг от друга. Анод --- плоскость, параллельная подложке. Для
вычисления распределения электростатического потенциала влияние
каждого эмиттера заменяется влиянием заряженной нити. При решении
граничной задачи для уравнения Пуассона используется метод
разделения переменных в~декартовых координатах. Распределение
потенциала найдено в~аналитическом виде во всей области системы.
В~соответствии с~полученными формулами проведены численные расчеты
и~показано влияние плот\-нос\-ти упаковки эмиттеров на
распределения
электростатического потенциала.\\[1mm]
\textit{Ключевые слова}: полевой катод, массив полевых эмиттеров, граничная задача,  электростатический потенциал, уравнение Пуассона, математическое моделирование.

}

}

{\bf Введение.} В~последние десятилетия активно разрабатываются
электронно\--ва\-ку\-умные приборы с~ низкоэнергетическими пучками
на основе полевой эмиссии [1--4]. Полевые катоды используются
в~электронных микроскопах, системах диагностики поверхности
методами неразрушающего контроля, индикаторах, плоских дисплеях
и~т.~д. [5--9].

В данной работе моделируется двумерная периодическая система
полевых эмиттеров. Эмиттеры  бесконечной длины и~одинаковой формы
расположены на плоской подложке. Анодом является плоскость,
параллельная плоскости подложки. Для вычисления распределения
электростатического потенциала во всей области системы
используется метод разделения переменных в~декартовых координатах
$(x,y)$. Схематическое изображение исследуемой системы эмиттеров
приведено на рис.~1.

%{\bf Постановка задачи.}
В силу периодичности исследуемой системы можно выделить ячейку,
содержащую один эмиттер. Не нарушая общность задачи, потенциал
эмиттера считаем равным нулю.   Для расчета распределения поля
в~ячейке заменим влияние эмиттера на поле влиянием заряженной нити
с линейной плотностью заряда $\tau(y)$, так что нулевая
эквипотенциаль совпадает с~поверхностью эмиттера  [10--12].

Параметры задачи:
$L$ --- высота полевого эмиттера;
$x_1$ ---  период системы по переменной $x$;
$y=0$, $0\le x \le x_1$ ---  плоскость подложки;
$y=y_1$, $0\le x \le x_1$ ---  плоскость анода;
$x_0=x_1/2$,  $0\le y \le y_0$  --- координаты заряженной нити;
$U(x,0)=0$ --- потенциал подложки;
$U(x,y_1)=U_0$ --- потенциал анода.




{\bf Математическая модель.}
Для определения распределения электростатического потенциала $U(x,y)$ во всей области элементарной ячейки ($0\le x \le x_1$, $0\le y \le y_1$) требуется решить граничную задачу для уравнения Пуассона:

\begin{eqnarray}
\left \{
\begin{array}{ll}
& \frac{\displaystyle \partial^2 U(x,y)}{\displaystyle \partial x^2}+
\frac{\displaystyle \partial^2 U(x,y)}{\displaystyle \partial y^2}
=-{\displaystyle 1\over \displaystyle \varepsilon_0}\rho(x,y),
\mbox{\rule[-3mm]{0mm}{6mm}}
\cr
&   U(x,0)=0,\ \ \ \ 0\le x \le x_1,
\mbox{\rule[-3mm]{0mm}{9mm}} \cr
&  U(x,y_1)=U_0, \ \ \ \ 0\le x \le x_1,
\mbox{\rule[-3mm]{0mm}{9mm}} \cr
&  {\displaystyle \partial  U(x,y)\over \displaystyle \partial x}
{\bigg \vert}_{\displaystyle {x=0}}=0, \ \ \ \ 0\le y \le y_1,
\mbox{\rule[-3mm]{0mm}{12mm}} \cr
&  {\displaystyle \partial  U(x,y)\over \displaystyle \partial x}
{\bigg \vert}_{\displaystyle {x=x_0}}=0, \ \ \ \ 0\le y \le y_1.
\mbox{\rule[-3mm]{0mm}{12mm}} \cr
\end{array}
\right.
\end{eqnarray}%1

Будем считать, что заряженная нить с~линейной плотностью заряда
$\tau(y)$ создает пространственное распределение заряда
$\rho(x,y)$ в~прямоугольнике малого объема ($|x-x_0|<\delta$,
$0\le y \le y_0$) [13, 14]:

\begin{eqnarray}
\rho(x,y)=
\left \{
\begin{array}{ll}
\rho(y)
, \ \ \ \  & |x-x_0|<\delta \ \mbox{{\rm и}}\ \ 0\le y \le y_0,
\mbox{\rule[-3mm]{0mm}{6mm}}
\cr
0, &  |x-x_0|>\delta \ \mbox{{\rm или}}\ \  y_0\le y \le y_1,
\end{array}
\right.
\end{eqnarray}%2
так что
\begin{eqnarray}
\tau(y)=\displaystyle\lim\limits_{\displaystyle \delta\to 0  } 2\delta \, y_0\,\rho(y).
\end{eqnarray}%3

%\vskip .3cm
{\bf Решение граничной задачи.} Представим решение
граничной задачи (1)--(3) в~виде
\begin{eqnarray}
    U(x,y)=U_0\,y/y_1+U_1(x,y).
\end{eqnarray}%4
Для определения функции $U_1(x,y)$ требуется решить уравнение
Пуассона с~однородными граничными условиями первого рода по
переменной $y$ и~однородными граничными условиями второго рода по
переменной $x$:

\begin{eqnarray}
\left \{
\begin{array}{ll}
& \frac{\displaystyle \partial^2 U_1(x,y)}{\displaystyle \partial x^2}+
\frac{\displaystyle \partial^2 U_1(x,y)}{\displaystyle \partial y^2}
=-{\displaystyle 1\over \displaystyle \varepsilon_0}\rho(x,y),
\mbox{\rule[-3mm]{0mm}{6mm}}
\cr
&   U_1(x,0)=0,\ \ \ \ 0\le x \le x_1,
\mbox{\rule[-3mm]{0mm}{9mm}} \cr
&  U_1(x,y_1)=0, \ \ \ \ 0\le x \le x_1,
\mbox{\rule[-3mm]{0mm}{9mm}} \cr
&  {\displaystyle \partial  U_1(x,y)\over \displaystyle \partial x}
{\bigg \vert}_{\displaystyle {x=0}}=0, \ \ \ \ 0\le y \le y_1,
\mbox{\rule[-3mm]{0mm}{12mm}} \cr
&  {\displaystyle \partial  U_1(x,y)\over \displaystyle \partial x}
{\bigg \vert}_{\displaystyle {x=x_0}}=0, \ \ \ \ 0\le y \le y_1.
\mbox{\rule[-3mm]{0mm}{12mm}} \cr
\end{array}
\right.
\end{eqnarray}%5

Правая часть уравнения Пуассона в~(5) определяется по формулам
(2), (3). Функцию $U_1(x,y)$ представим как ряд по собственным
функциям по переменной $x$ с~коэф\-фи\-циен\-та\-ми $v_{n}(y)$
[15, 16]:
\begin{eqnarray}
U_1(x,y)=\displaystyle\sum\limits_{n=0}^{\infty}
v_{n}(y)\cos {\alpha_n x},\ \ \
\alpha_n=\displaystyle {\pi n \over \displaystyle x_1}.
\end{eqnarray}%6
Тогда, подставляя выражение (6) в~уравнение Пуассона (5), и~из
ортогональности собственных функций получим, что каждая функция
$v_{n}(y)$ удовлетворяет диф\-фе\-рен\-циаль\-но\-му уравнению
\begin{eqnarray}
v_{n}^{''}(y)-
\left(\alpha_{n}\right)^2 v_{n}(y)=\dfrac{\gamma_{n}}{x_1}\phi_{n}(y),
\ \ \ \gamma_{n}=\left\{
\begin{array}{ll}
1, \  n=0,&  \cr 2, \ n>0,&
\end{array}
\right.
\end{eqnarray}%7
где
\begin{eqnarray}
\phi_{n}(y)=
-{\displaystyle 1\over \displaystyle \varepsilon_0}
\int\limits_{0}^{x_1}
\rho(x,y)\cos {\alpha_n x},
\end{eqnarray}%8
а однородность граничных условий для $U_1(x,y)$ в~(5) по
переменной $y$ ведет к~однородности граничных условий для функций
$v_{n}(y)$:
\begin{eqnarray}
v_{n}(0)=v_{n}(y_1)=0, \ \ \ n\ge 0.
\end{eqnarray}%9
Функции $\phi_{n}(y)$ уравнения (8) в~соответствии с~формулами
(2), (3) вычисляются в~явном виде
\begin{eqnarray}
\phi_{n}(y)=
\left \{
\begin{array}{ll}
-{\displaystyle 1\over \displaystyle \varepsilon_0}\,
{\displaystyle 2\delta \rho(y) \cos {\alpha_n x_0}}=-{\displaystyle 1\over \displaystyle \varepsilon_0\,y_0}\,
{\displaystyle \tau(y)}\cos {\alpha_n x_0},
&  0\le y\le y_0,
\mbox{\rule[-3mm]{0mm}{6mm}}
\cr
\hspace*{0.1cm}0, &   y_0<y\le y_1.
\end{array}
\right.
\end{eqnarray}%10

%\vskip 7mm

Решая дифференциальное уравнение (7) с~правой частью (10)
и~граничными условиями (9), получим

%\vskip 7mm

при $n=0$
\begin{equation}
\begin{array}{ll}
\displaystyle v_0(y) =- \dfrac{y_1-y}{x_1\,y_1}
\int\limits_{0}^{y}\eta\phi_0(\eta)\,d\eta+
\dfrac{y}{x_1\,y_1}
\int\limits_{y}^{y_0}(\eta-y_1)\phi_0(\eta)\,d\eta, & 0\le y\le y_0,
\\[2mm]
\displaystyle v_0(y) =- \dfrac{y_1-y}{x_1\,y_1}
\int\limits_{0}^{y_0}\eta\phi_0(\eta)\,d\eta, & y_0<y\le y_1,
\end{array}
\end{equation}%11

%\vskip 7mm

при $n>0$
\begin{equation}
\begin{array}{ll}
\displaystyle v_n(y) =  - \dfrac{2}{\pi n }
\dfrac{1}{\sinh\alpha_n y_1}
\Bigg[\int\limits_{0}^{y}\phi_n(\eta)\sinh\alpha_n \eta\sinh\alpha_n (y_1-y)\,d\eta \ -
& \\[2mm]
\hspace*{3.5cm}  \displaystyle
-\int\limits_{y}^{y_0}\phi_n(\eta)\sinh\alpha_n (\eta-y_1)\sinh\alpha_n y\,d\eta
\Bigg],  & 0\le y\le y_0,
 \\[2mm]
\displaystyle v_n(y) =  - \dfrac{2}{\pi n }
\dfrac{1}{\sinh\alpha_n y_1}
\Bigg[\int\limits_{0}^{y_0}\phi_n(\eta)\sinh\alpha_n
\eta\sinh\alpha_n (y_1-y)\,d\eta\Bigg], & y_0<y\le y_1.
\end{array}
\end{equation}%12

%\vskip 7mm

Итак, решение исходной граничной задачи   (1)--(3), согласно
формулам (4),  (6), (10)--(12), получено в~аналитическом виде во
всей области исследуемой диодной системы.


%\vskip .7cm
{\bf  Результаты численных расчетов.}
Для проведения вычислений примем, что функция, задающая плотность заряда $\tau(y)$, является линейной [15, 16]:
$$
\tau(y)=\tau\,y/y_0,
$$
где $\tau=$const.



Распределение электростатического потенциала $U(x,y)$  (см.
формулы  (4), (6), (10)--(12)) можно представить в~явном виде:

%\vskip .3cm

при $0\le y\le y_0$
%
\begin{equation}
\begin{array}{l}
\displaystyle U(x,y) =  U_0 \dfrac{y}{y_1}+\dfrac{\tau}{\varepsilon_0 y_0^2}
\Bigg[\dfrac{1}{6x_1\,y_1}\left( 3 y\, y_0^2 y_1-2y\, y_0^3-y^3 y_1     \right)+
\\[.8mm]
\hspace*{0.2cm}  \displaystyle +
\dfrac{2}{x_1}\sum\limits_{n=0}^{\infty}(-1)^n
\dfrac{1}{\alpha_{2n}^2\sinh\alpha_{2n} y_1} \Big( y
\sinh\alpha_{2n} y_1 -y_0 \sinh\alpha_{2n} y \cosh\alpha_{2n}
(y_1-y_0) -
\\[.8mm]
\hspace*{3.5cm}  \displaystyle -\,
\alpha_{2n}^{-1}\sinh\alpha_{2n} y \sinh\alpha_{2n} (y_1-y_0)
\Big) \Bigg] \cos {\alpha_{2n} x},
\end{array}
\end{equation}%13

%\vskip .3cm

при $y_0<y\le y_1$
%
\begin{equation}
\begin{array}{l}
\displaystyle U(x,y) =  U_0
\dfrac{y}{y_1}+\dfrac{\tau}{\varepsilon_0 y_0^2}
\Bigg[\dfrac{1}{3x_1\,y_1}(y_1-y)y_0^3 \,+
\\[.7mm]
\hspace*{0.8cm}  \displaystyle +\,
\dfrac{2}{x_1}\sum\limits_{n=0}^{\infty}(-1)^n
\dfrac{\sinh\alpha_{2n} (y_1-y)}{\alpha_{2n}^2\sinh\alpha_{2n}
y_1} \Big( y_1 \cosh\alpha_{2n} y_0 -
\alpha_{2n}^{-1}\sinh\alpha_{2n} y_0  \Big) \Bigg] \cos
{\alpha_{2n} x}.
\end{array}
\end{equation}%14

%\vskip .7cm

В соответствии с~полученным аналитическим решением граничной
задачи (1)--(3) были произведены численные расчеты. Для вычисления
потенциала $U(x,y)$ требуется найти величину $\tau$, входящую
в~формулы (13), (14). Для этого решается уравнение\pagebreak

$$
U(x_0,L)=0,
$$
где $L=y_0+\varepsilon$.

Все значения параметров, как геометрических, так и~потенциалов,
берутся в~безразмерных величинах.

{\bf Заключение.} В~данной статье моделируется двумерная
эмиссионная диодная система, в~которой полевым катодом является
периодическая система эмиттеров одинаковой формы, расположенных на
плоской подложке, и~с~плоским анодом, параллельным подложке. При
вычислении электростатического распределения потенциала влияние
эмиттеров на поле заменяется влиянием системы заряженных нитей.
Для решения поставленной граничной задачи (1)--(3) применяется
метод разделения переменных в декартовых координатах.
Распределение потенциала с~учетом периодичности исследуемой
системы найдено в~виде рядов с~известными коэффициентами ---
формулы (4), (6), (10)--(12). Для численных расчетов
электростатического потенциала по формулам (13), (14) была
использована линейная функция, задающая плотность заряда нити.
Представленные графики показывают зависимость распределения
потенциала от расстояния между соседними эмиттерами, что
согласуется с~известными экспериментальными фактами.
