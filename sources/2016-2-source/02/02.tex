OF THE LYAPUNOV MATRIX FOR A CLASS OF TIME \\ DELAY SYSTEMS}

\vskip2mm


{\footnotesize


\noindent St.~Petersburg State University, 7--9, Universitetskaya
nab.,\\ St.~Petersburg, 199034, Russian Federation

\vskip2.5mm

\begin{list}{}{\leftmargin=7mm \rightmargin=7mm \listparindent=5mm}

\item \hskip5mm The Lyapunov matrix for systems of linear time-delay equations is
a matrix-valued function which is a solution of a special dynamic
system with some additional boundary conditions. This matrix
allows to construct the complete type Lyapunov--Krasovskii
functionals with a prescribed derivative, which are used
successfully in analysis of behavior of time-delay systems. It was
shown in works of Kharitonov and Chashnikov that the Lyapunov
condition, that is the absence of opposite eigenvalues of the
system, guarantees the existence and uniqueness of the Lyapunov
matrix for systems of retarded type with multiple delays and for
systems of neutral type with a single delay. In this contribution,
we consider a linear time-invariant system of neutral type with
two delays. It is shown that under a certain constraint the
Lyapunov condition, that is the absence of opposite eigenvalues of
the system, is also a criterion of the existence and uniqueness of
the Lyapunov matrix. Refs 14.

\textit{Keywords}: time-delay system, neutral type equation,
Lyapunov matrix.

\end{list}

}

\textbf{1. Введение.} Метод функционалов
Ляпунова--Красовского~[1--3]~--- один из~наиболее популярных при
исследовании устойчивости систем с~запаздыванием. Он обобщает
второй (или прямой) метод Ляпунова, широко используемый для систем
обыкновенных дифференциальных уравнений. Одним из~фундаментальных
различий между системами с~запаздыванием и системами без
запаздывания является то, что у~первых состоянием служит функция,
а~у~последних~--- точка. Поэтому Н.~Н.~Красовским~[1] было
предложено вместо функций Ляпунова применять функционалы. Было
доказано, что достаточно подобрать положительно определенный
функционал, производная которого вдоль решений системы
отрицательно определена, чтобы установить экспоненциальную
устойчивость системы с~запаздыванием.

Для исследования устойчивости линейных стационарных систем
обыкновенных дифференциальных уравнений используют функции вида
$v(x)=x^T V x$, где $V$~--- матрица Ляпунова. Эта матрица является
решением известного алгебраического матричного уравнения и~может
быть найдена, например, методом Гаусса. Критерием существования
и~единственности решения является условие Ляпунова~--- отсутствие
у~системы собственных чисел, симметричных относительно нуля
комплексной плоскости.

Для разных классов систем уравнений с~запаздыванием в~[3--5] была
введена матрица Ляпунова, которая представляет собой функцию
вещественной переменной. С~помощью матриц Ляпунова можно построить
функционалы полного типа, которые успешно применяются для изучения
устойчивости и~определения различных параметров систем
с~запаздыванием (см.~[3,\,6--10]).

Матрица Ляпунова является решением довольно сложной системы
динамических и~алгебраических уравнений. Естественно, возникает
вопрос построения этой матрицы. Для систем запаздывающего типа
теория матриц Ляпунова к~настоящему времени развита достаточно
хорошо: для случая кратных запаздываний метод построения матриц
Ляпунова был найден, о~нем можно прочесть, например, в~книге~[3],
для некратных запаздываний существует ряд численных
процедур~[3,\,8,\,11,\,12]. Кроме того, для систем с~кратными~[13]
и систем с~некратными запаздываниями~[3] было доказано, что
критерий существования и~единственности матрицы Ляпунова~--- то же
самое условие, что и~для систем без запаздывания~--- условие
Ляпунова.

В этой статье речь пойдет о~системах нейтрального типа. Для случая
одного запаздывания в~системе критерий существования
и~единственности матрицы Ляпунова был доказан в~[3]. Однако,
с~одной стороны, примененный метод доказательства не~допускает
обобщения для систем с~несколькими запаздываниями. С~другой
стороны, в~[13] доказательство дано для систем запаздывающего типа
с~несколькими кратными запаздываниями, но оно не~может быть
использовано для систем нейтрального типа. В~данной работе введено
понятие обобщенных матриц Ляпунова и~получен ряд их свойств,
которые позволили предложить новый метод доказательства критерия
существования и~единственности матрицы Ляпунова, применимый для
систем нейтрального типа с~несколькими запаздываниями. Мы решили
ограничиться двумя запаздываниями исключительно ради наглядности
большинства формул, хотя результаты, представленные в~статье,
могут быть перенесены на~случай любого конечного числа кратных
запаздываний.

Структура статьи следующая. В~п.~2 вводится система; п.~3 посвящен
ее спектру. В~п.~4 предлагается понятие обобщенной матрицы
Ляпунова и~доказываются некоторые ее свойства. Алгоритм построения
обобщенной матрицы Ляпунова описан в~п.~5. В~п.~6 отдельно
рассматривается тривиальный случай, когда обобщенная матрица
Ляпунова ассоциирована с~нулевой матрицей $W$. В~п.~7 представлен
основной\linebreak\newpage\noindent результат~--- критерий
существования и~единственности матрицы Ляпунова. Краткое
заключение (см. п.~8) завершает статью.

\textbf{2. Общие сведения.} Будем рассматривать линейную систему
нейтрального \mbox{типа}
\begin{gather}
\frac{d}{dt}\left[x(t)+D_1 x(t-h)+D_2 x(t-2h)\right]= \notag\\
=A_0 x(t)+A_1 x(t-h)+A_2 x(t-2h),\quad t>0,~a.~e., \label{systz}
\end{gather}
в которой $x(t)\in \R^n$, $D_1,\,D_2,\,A_0,\,A_1,\,A_2$~---
вещественные матрицы порядка $n\times n$, запись $a.~e.$ (almost
everywhere) означает, что равенство выполнено почти всюду. Система
содержит два положительных кратных запаздывания: $h$ и~$2h$. Для
краткости записи будем использовать матрицу $D_0=I$ (единичную
матрицу). В~дальнейшем существенным будет следующее предположение:
\begin{ass} \label{hypAss}\label{hypSyst}\hskip -1.5mm{\bf.} Система
\begin{equation}
\begin{split}
& Y_1+Y_2\, D_1+Y_3\, D_2=0,\\
& Y_2+Y_3\, D_1+Y_4\, D_2=0,\\
& Y_3+D_1^T\, Y_2+D_2^T\, Y_1=0,\\
& Y_4+D_1^T\, Y_3+D_2^T\, Y_2=0,
\end{split}
\end{equation}
где $Y_1,\,Y_2,\,Y_3,\,Y_4\in \R^{n\times n}$, имеет только
нулевое решение.
\end{ass}

Для решения систем матричных алгебраических уравнений (см. (2))
часто применяют метод векторизации. Введем операцию векторизации,
которая ставит матрице $X=(X_1,\ldots,X_n)\in\R^{n\times n}$
в~соответствие вектор, составленный из~столбцов этой матрицы:
$$
\vec(X)=
\begin{pmatrix}
X_1\\
\ldots\\
X_n
\end{pmatrix}
\in\R^{n^2}.
$$
Отметим, что для $A$, $B\in\R^{n\times n}$
$$
\vec(A\cdot X\cdot B)=A\otimes B \cdot \vec(X).
$$
Матрицу
$$
A\otimes B=
\begin{pmatrix}
b_{11}A & b_{21}A & \ldots & b_{n1}A\\
b_{12}A & b_{22}A & \ldots & b_{n2}A\\
\ldots & \ldots & \ldots & \ldots\\
b_{1n}A & b_{2n}A & \ldots & b_{nn}A
\end{pmatrix}
\in\R^{n^2\times\,n^2}
$$
называют \emph{произведением Кронекера}, где $b_{ij}$~---
соответствующие компоненты матрицы~$B$.

Векторизовав систему~\eqref{hypSyst}, получим очевидное
утверждение.

\begin{lem}\hskip -1.5mm{\bf.}
\label{hypAssEq} Предположение~\textup{\ref{hypAss}} выполнено
тогда и~только тогда, когда матрица
$$
M_1=
\begin{pmatrix}
I\otimes I & I\otimes D_1 & I\otimes D_2 & 0 \\
0 & I\otimes I & I\otimes D_1 & I\otimes D_2 \\
D_2^T\otimes I & D_1^T\otimes I & I\otimes I & 0 \\
0 & D_2^T\otimes I & D_1^T\otimes I & I\otimes I
\end{pmatrix}
$$
невырождена.
\end{lem}


\textbf{3. Спектр системы.} \emph{Спектром} $\Lambda$
системы~\eqref{systz} называют множество ее \emph{собственных
чисел}, т.~е. нулей характеристического квазиполинома
$$
f(s)=\det\left(sI+s\,e^{-hs}D_1+s\,e^{-2hs}D_2-A_0-e^{-hs}
A_1-e^{-2hs} A_2\right).
$$
Как было показано в~[14], расположение на~комплексной плоскости
бесконечно больших по~модулю собственных чисел определяется нулями
полинома
$$
g(z)=\det\left(z^2I+z D_1+D_2\right),
$$
который соответствует разностной системе
\begin{equation}
\label{systDiff} y(t)+D_1 y(t-h)+D_2y(t-2h)=0
\end{equation}
в том смысле, что разностная система экспоненциально устойчива
тогда и~только тогда, когда все нули $g(z)$ лежат внутри
единичного круга комплексной плоскости.

Каждому нулю $z_0$ полинома $g(z)$ соответствует бесконечная
цепочка нулей функции $f(s)$, расположенных вдоль прямой
$\text{Re}(s)=\ln|z_0|/h$. Поэтому необходимым условием
экспоненциальной устойчивости системы~\eqref{systz} является
экспоненциальная устойчивость системы~\eqref{systDiff}.

Цель данной работы~--- показать, что критерием существования
и~единственности матрицы Ляпунова для системы~\eqref{systz}
служит, как и~в~случае уравнений запаздывающего типа, выполнение
условия Ляпунова.

\begin{defin}\hskip -1.5mm{\bf.}
\label{LyapCondDef} Для системы~\eqref{systz} выполняется
\textit{условие Ляпунова}, если существует число $\varepsilon>0$
такое, что $|s_1+s_2|>\varepsilon$ для любых чисел $s_1$, $s_2$,
равных нулям характеристического квазиполинома $f(s)$.
\end{defin}

Как было отмечено в п.~1, для систем запаздывающего типа условием
Ляпунова является отсутствие собственных чисел, дающих в~сумме
нуль. Представленное в~определении условие Ляпунова дополнительно
гарантирует, что не~существует двух последовательностей
собственных чисел, пределы которых дают в~сумме нуль. Для
уравнений запаздывающего типа этого в~принципе не~может быть, и,
как показывает сле\-дую\-щая лемма, доказательство которой подобно
доказательству леммы 6.8 из~книги~[3], с~ней мы также
не~столкнемся из-за введенного ранее предположения~\ref{hypAss}.

\begin{lem}\hskip -1.5mm{\bf.}
\label{LyapCond} Пусть выполнено
предположение~\textup{\ref{hypAss}}. Система нейтрального
типа~\textup{\eqref{systz}} удовлетворяет условию Ляпунова тогда
и~только тогда, когда никакие числа $s_0$ и~$-s_0$ не~являются
одновременно собственными числами данной системы.
\end{lem}
\begin{proof}
\textit{Необходимость} очевидна. Докажем \textit{достаточность}.
Предположим, что условие Ляпунова не~выполнено. Следовательно,
существуют две последовательности собственных чисел системы
$\left\{s_1^{(k)}\right\}$, $\left\{s_2^{(k)}\right\}$, такие что
$$
\lim_{k\rightarrow \infty}\left(s_1^{(k)}+s_2^{(k)}\right)=0.
$$
А следовательно,
$$
\lim_{k\rightarrow \infty}\text{Re}
\left(s_1^{(k)}\right)=-\lim_{k\rightarrow \infty}\text{Re}
\left(s_2^{(k)}\right)=r,
$$
где $r$~--- некоторое конечное число (системы нейтрального типа
не~имеют бесконечных цепочек корней, уходящих вправо). Так как
квазиполином~--- аналитическая функция, в~любой ограниченной
области функция $f(s)$ имеет конечное число нулей. Потому можно
утверждать, что
$$
\lim_{k\rightarrow
\infty}\left|s_1^{(k)}\right|=\lim_{k\rightarrow
\infty}\left|s_2^{(k)}\right|=\infty.
$$
Как было показано в~[14], числа
$$
z_1=\lim_{k\rightarrow \infty}e^{hs_1^{(k)}},\quad
z_2=\lim_{k\rightarrow \infty}e^{hs_2^{(k)}}
$$
являются корнями полинома $g(z)$. При этом
$$
z_1z_2=\lim\limits_{k\rightarrow
\infty}e^{h\left(s_1^{(k)}+s_2^{(k)}\right)}=1.
$$
Докажем теперь, что это противоречит предположению~\ref{hypAss}.
Действительно, для $z_1$ и~$z_2$ найдутся ненулевые векторы $p$
и~$q$ такие, что
\begin{gather*}
\left(z_1^2I+z_1 D_1+D_2\right)^Tp=0,\\
\left(z_2^2I+z_2 D_1+D_2\right)^Tq=z_2^2\left(I+z_1
D_1+z_1^2D_2\right)^Tq=0.
\end{gather*}
Легко показать теперь, что набор матриц
$$
Y_1=p\,q^T,\quad Y_2=z_1\,p\,q^T,\quad Y_3=z_1^2\,p\,q^T,\quad
Y_4=z_1^3\,p\,q^T
$$
является ненулевым решением системы~\eqref{hypSyst}.
\end{proof}


\textbf{4. Матрица Ляпунова и~обобщенная матрица Ляпунова.}
В~работе~[5] было дано определение матрицы Ляпунова для систем
нейтрального типа.

\begin{defin}\hskip -1.5mm{\bf.}
\label{MatrL} Для системы~\eqref{systz} \emph{матрицей Ляпунова},
ассоциированной с~симметричной постоянной матрицей $W$, будем
называть непрерывную функциональную матрицу $U(\tau)$,
$\tau\in\R$, удовлетворяющую уравнениям
\begin{gather*}
U'(\tau)+U'(\tau-h) D_1+U'(\tau-2h) D_2= \\
= U(\tau) A_0+U(\tau-h) A_1+U(\tau-2h) A_2, \quad \tau>0,~a.~e., \\
U(-\tau)=U^T(\tau),\quad \tau\in\R,\\
\sum_{j=0}^2\sum_{k=0}^2\left[D_k^TU\left( \vph
(k-j)h\right)A_j+A_k^TU \left(\vph(k-j)h\right) D_j\right]=-W.
\end{gather*}
Первое уравнение называется \emph{динамическим свойством},
второе~--- \emph{свойством симметрии}, а~третье~---
\emph{алгебраическим свойством}.
\end{defin}

Из определения~\ref{MatrL} можно вывести еще одно свойство матрицы
Ляпунова.

\begin{lem}\hskip -1.5mm{\bf.}
Матрица Ляпунова удовлетворяет свойству
\begin{gather*}
U'(\tau)+D_1^T U'(\tau+h)+D_2^TU'(\tau+2h)=\\
= -A_0^T U(\tau)-A_1^TU(\tau+h)-A_2^TU(\tau+2h),\quad
\tau<0,~a.~e.
\end{gather*}
\end{lem}
\begin{proof}
Из свойства симметрии следует, что
$$
U'(\tau)=-\left[U'(-\tau)\right]^T,\quad \tau\in\R,~a.~e.
$$
Транспонируем обе части динамического свойства и~применим к~нему
получен-\linebreak ное равенство и~само свойство симметрии.
Заменив $-\tau$ на~$\tau$, завершим доказа\-тельство.
\end{proof}

Введем теперь новое определение.

\begin{defin}\hskip -1.5mm{\bf.}
\label{MatrLgen} Для системы~\eqref{systz} \emph{обобщенной
матрицей Ляпунова}, ассоциированной с~симметричной постоянной
матрицей $W$, будем называть непрерывную функциональную матрицу
$U_0(\tau)$, $\tau\in\R$, удовлетворяющую уравнениям
\begin{gather}
U_0'(\tau)+U_0'(\tau-h) D_1+U_0'(\tau-2h) D_2= \notag\\
= U_0(\tau) A_0+U_0(\tau-h) A_1+U_0(\tau-2h) A_2, \quad \tau>0,~a.~e., \label{svD0}\\
U_0'(\tau)+D_1^T U_0'(\tau+h)+D_2^TU_0'(\tau+2h)= \notag\\
= -A_0^T U_0(\tau)-A_1^TU_0(\tau+h)-A_2^TU_0(\tau+2h),\quad \tau<0,~a.~e., \label{svDneg0}\\
\sum_{j=0}^2\sum_{k=0}^2\left[D_k^TU_0\left( \vph
(k-j)h\right)A_j+A_k^TU_0 \left(\vph(k-j)h\right) D_j\right]=-W.
\label{svA0}
\end{gather}
\end{defin}

Определения~\ref{MatrL} и~\ref{MatrLgen} не~являются
эквивалентными (хотя, как будет показано в~п.~7, они эквивалентны,
если выполнено условие Ляпунова), так как обобщенная матрица
Ляпунова, вообще говоря, не~удовлетворяет свойству симметрии. Но
можно заметить, что если $U_0(\tau)$ есть обобщенная матрица
Ляпунова, то и~$U_0^T(-\tau)$ тоже ею является.

\begin{lem}\hskip -1.5mm{\bf.}
\label{GenToOrd} Если $U_0(\tau)$~--- обобщенная матрица Ляпунова,
ассоциированная с~$W$, то
$$
U(\tau)=\dfrac{1}{2}\left(U_0(\tau)+U_0^T(-\tau)\right)
$$
является матрицей Ляпунова, ассоциированной с~той же $W$.
\end{lem}

Очевидно, что любая матрица Ляпунова есть обобщенная матрица
Ляпунова, поэтому все свойства, которые будут получены, относятся
как к~$U_0(\tau)$, так и~к~$U(\tau)$.

\begin{lem}\hskip -1.5mm{\bf.}
Если выполнено предположение~\textup{\ref{hypAss}}, то
односторонние производные обобщенной матрицы Ляпунова определены
в~каждой точке $\tau\in\R$.
\end{lem}
\begin{proof}
Исследуем первую производную обобщенной матрицы Ляпунова. Возьмем
произвольное $\tau\in(0,h]$. Построим уравнения:
\begin{itemize}\begin{itemize}
  \item динамическое свойство~\eqref{svD0} в~точке $\tau+h-\varepsilon$;
  \item динамическое свойство в~точке $\tau-\varepsilon$;
  \item свойство~\eqref{svDneg0} в~точке $\tau-h-\varepsilon$;
  \item свойство~\eqref{svDneg0} в~точке $\tau-2h-\varepsilon$.
\end{itemize}\end{itemize}
Здесь $\varepsilon>0$~--- достаточно малое число, такое что все
указанные выше подстановки корректны. Получаем систему
\begin{equation*}
\begin{split}
& Z_1'(\tau)+Z_2'(\tau) D_1+Z_3'(\tau) D_2= Z_1(\tau) A_0+Z_2(\tau)A_1+Z_3(\tau) A_2,\\
& Z_2'(\tau)+Z_3'(\tau) D_1+Z_4'(\tau)D_2= Z_2(\tau) A_0+Z_3(\tau)A_1+Z_4(\tau)A_2,\\
& Z_3'(\tau)+D_1^TZ_2'(\tau)+D_2^TZ_1'(\tau)=-A_0^T Z_3(\tau)-A_1^TZ_2(\tau)-A_2^TZ_1(\tau),\\
& Z_4'(\tau)+D_1^TZ_3'(\tau)+D_2^TZ_2'(\tau)=-A_0^T
Z_4(\tau)-A_1^TZ_3(\tau)-A_2^TZ_2(\tau),
\end{split}
\end{equation*}
где
$$
Z_1(\tau)=U_0(\tau+h-\varepsilon),\;~
Z_2(\tau)=U_0(\tau-\varepsilon),
$$
$$
Z_3(\tau)=U_0(\tau-h-\varepsilon),\;~
Z_4(\tau)=U_0(\tau-2h-\varepsilon).
$$
Так как предположение~\ref{hypAss} выполнено, система может быть
разрешена относительно производной. При этом в~правой части будут
стоять некоторые конечные величины (так как обобщенная матрица
Ляпунова непрерывна по~определению). Устремив $\varepsilon$ к~нулю
так, чтобы сохранить корректность четырех вышеназванных
подстановок, получим, что левые производные матрицы $U_0(\tau)$
в~точках $\tau+h$, $\tau$, $\tau-h$ и~$\tau-2h$ конечны. Таким
образом, мы доказали, что на~интервале $(-2h,\,2h]$ левая
производная обобщенной матрицы Ляпунова всюду определена. Докажем,
что это верно и~в~остальных точках. Например, в~точке
$\tau\in(2h,3h]$ левая производная равна
\begin{gather*}
U_0'(\tau-0)=-U_0'(\tau-h-0) D_1-U_0'(\tau-2h-0) D_2 ~+\\
+~U_0(\tau) A_0+U_0(\tau-h) A_1+U_0(\tau-2h) A_2.
\end{gather*}
Она конечна, так как конечны все величины в~правой части
равенства. Для остальных $\tau$ нужно действовать аналогично,
и~аналогично же можно доказать, что правая производная обобщенной
матрицы Ляпунова всюду определена.

Теперь можно продифференцировать (точнее, взять односторонние
производные) уравнения~\eqref{svD0} и~\eqref{svDneg0} и~доказать
все то же самое для второй производной, а~затем и~для производных
более высокого порядка.
\end{proof}

Таким образом, показано, что функция
$$
\Delta U_0'(\tau)=\lim_{\theta\rightarrow \tau+0} U_0'(\theta)
-\lim_{\theta\rightarrow \tau-0}
U_0'(\theta)=U_0'(\tau+0)-U_0'(\tau-0)
$$
определена при всех $\tau\in\R$.

\begin{lem}\hskip -1.5mm{\bf.}
\label{lemDelta} Имеют место следующие свойства$:$

\begin{enumerate}\begin{enumerate}
\item[$1)$] $\Delta U_0'(\tau)+\Delta U_0'(\tau-h)\,D_1+\Delta U_0'(\tau-2h)\,D_2 =0,\quad \tau>0;$
\item[$2)$] $\Delta U_0'(\tau)+D_1^T\Delta U_0'(\tau+h)+D_2^T\Delta U_0'(\tau+2h) =0,\quad \tau<0;$
\item[$3)$] алгебраическое свойство~\textup{\eqref{svA0}} может быть переписано следующим образом$:$ $\Delta U_0'(0)+\Delta U_0'(-h)D_1+\Delta U_0'(-2h)D_2 =-W$.
\end{enumerate}\end{enumerate}
\end{lem}
\begin{proof}
Первые два свойства вытекают из~непрерывности обобщенной матрицы
Ляпунова и~из~равенств~\eqref{svD0} и~\eqref{svDneg0}.

Кроме того, непрерывность обобщенной матрицы Ляпунова
и~свойства~\eqref{svD0},~\eqref{svDneg0} позволяют преобразовать
алгебраическое свойство~\eqref{svA0} к~виду
$$
\sum_{j=0}^2\sum_{k=0}^2 D_k^TU_0'\left(\vph(k-j)h+0\right)D_j
-\sum_{j=0}^2\sum_{k=0}^2 D_k^T
U_0'\left(\vph(k-j)h-0\right)D_j=-W.
$$
Равенство, полученное в~п.~1), приводит к~искомому свойству

$\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad\quad
\sum\limits_{j=0}^2 D_0^T\,\Delta U_0'\left(-jh\right)D_j=-W. $
\end{proof}

Представленные результаты позволяют продолжить исследование
аналитических свойств обобщенных матриц Ляпунова.

\begin{thm}\hskip -1.5mm{\bf.}
\label{thmAnalyt} Пусть выполнено
предположение~\textup{\ref{hypAss}}. Производные обобщенной
матрицы Ляпунова могут иметь разрывы только в~точках вида $k h$,
где $k$~--- целое число.
\end{thm}
\begin{proof}
Как мы теперь знаем, функция $\Delta U_0'(\tau)$ определена
во~всех точках $\tau\in\R$. Производная обобщенной матрицы
Ляпунова существует только в~тех точках, где $\Delta
U_0'(\tau)=0$.

Зафиксируем произвольное $\tau\in(0,h)$. По~лемме~\ref{lemDelta}
\begin{gather}
\Delta U_0'(\tau+h)+\Delta U_0'(\tau)D_1+\Delta U_0'(\tau-h)D_2=0, \label{DeltaEq1}\\
\Delta U_0'(\tau)+\Delta U_0'(\tau-h)D_1+\Delta U_0'(\tau-2h)D_2=0, \label{DeltaEq2}\\
\Delta U_0'(\tau-h)+D_1^T\Delta U_0'(\tau)+D_2^T\Delta U_0'(\tau+h)=0, \label{DeltaEq3}\\
\Delta U_0'(\tau-2h)+D_1^T\Delta U_0'(\tau-h)+D_2^T\Delta
U_0'(\tau)=0. \label{DeltaEq4}
\end{gather}
В итоге получим систему уравнений, из~которой, благодаря
предположению~\ref{hypAss}, вы\-те\-кают существование
и~непрерывность первой производной во~всех точках
$\tau\in(-2h,-h)\cup(-h,0)\cup(0,h)\cup(h,2h)$. Для остальных
интервалов между точками, кратными $h$, доказать непрерывность
первой производной не~составит труда. А~вот доказать непрерывность
в точках, кратных $h$, не~удастся, так как, вообще говоря,
$$
\Delta U_0'(0)+\Delta U_0'(-h)D_1+\Delta U_0'(-2h)D_2=-W\neq0.
$$

Продифференцировав уравнения~\eqref{svD0} и~\eqref{svDneg0}, можно
доказать то же самое для второй производной, а~затем и~для
производных более высокого порядка.
\end{proof}


\textbf{5. Построение обобщенной матрицы Ляпунова.} Если мы
определим обобщенную матрицу Ляпунова на~отрезке $[-2h,\,2h]$, то
сможем единственным образом продолжить ее на~всю вещественную
прямую, применив метод шагов~[14] к~динамическому
свойству~\eqref{svD0}. Поэтому опишем метод построения матрицы
на~отрезке $[-2h,2h]$.

Пусть снова выполнено предположение~\ref{hypAss}. Каждой
обобщенной матрице Ляпунова $U_0(\tau)$ можно поставить
в~соответствие матрицы
$$
Z_1(\tau)=U_0(\tau+h),\;~Z_2(\tau)=U_0(\tau),\;~
Z_3(\tau)=U_0(\tau-h),\;~ Z_4(\tau)=U_0(\tau-2h),
$$
которые удовлетворяют системе
\begin{equation}
\label{DelayFreeSyst}
\begin{split}
& Z_1'(\tau)+Z_2'(\tau) D_1+Z_3'(\tau) D_2= Z_1(\tau) A_0+Z_2(\tau)A_1+Z_3(\tau) A_2,\\
& Z_2'(\tau)+Z_3'(\tau) D_1+Z_4'(\tau)D_2= Z_2(\tau) A_0+Z_3(\tau)A_1+Z_4(\tau)A_2,\\
& Z_3'(\tau)+D_1^TZ_2'(\tau)+D_2^TZ_1'(\tau)=-A_0^T Z_3(\tau)-A_1^TZ_2(\tau)-A_2^TZ_1(\tau),\\
& Z_4'(\tau)+D_1^TZ_3'(\tau)+D_2^TZ_2'(\tau)=-A_0^T
Z_4(\tau)-A_1^TZ_3(\tau)-A_2^TZ_2(\tau).
\end{split}
\end{equation}
Первые два равенства получены из~динамического свойства,
а~последние~--- из~свойства~\eqref{svDneg0}. Равенства верны для
$\tau\in[0,h]$ (если в~точке $\tau=0$ использовать правую
производную, а~в~точке $\tau=h$~--- левую).

Дополним систему~\eqref{DelayFreeSyst} набором граничных условий
\begin{equation}
\label{DelayFreeSystBound}
\begin{split}
& Z_1(0)=Z_2(h),\\
& Z_2(0)=Z_3(h),\\
& Z_3(0)=Z_4(h),\\
& D_1^T Z_1(0)A_0 +D_2^T Z_1(0)A_1 +A_1^T Z_1(0) +A_2^T Z_1(0)D_1+Z_2(0)A_0~+\\
& \hphantom{a} +D_1^T Z_2(0)A_1 +D_2^T Z_2(0)A_2 +A_0^T Z_2(0) +A_1^T Z_2(0)D_1 +A_2^T Z_2(0)D_2~+\\
& \hphantom{aa}+Z_3(0)A_1 +D_1^T Z_3(0)A_2 +A_0^T Z_3(0)D_1 +A_1^T Z_3(0)D_2 ~+\\
& \hphantom{aaa}+Z_4(0)A_2 +A_0^T Z_4(0)D_2+D_2^T Z_1(h) A_0+A_2^T
Z_1(h)=-W.
\end{split}
\end{equation}
Первые три уравнения вытекают из~непрерывности обобщенной матрицы
Ляпунова, а~последнее~--- из~алгебраического свойства~\eqref{svA0}
с учетом того, что
$$
U_0'(-2h-0)=-D_1^T U_0'(-h-0)-D_2^TU_0'(-0) -A_0^T
U_0(-2h)-A_1^TU_0(-h)-A_2^TU_0(0).
$$

Система~\eqref{DelayFreeSyst} может быть переписана
в~векторизованной форме $M_1 z'(\tau)=M_2 z(\tau)$, где
$$
z(\tau)=\vec\left(Z_1(\tau)\;Z_2(\tau)
\;Z_3(\tau)\;Z_4(\tau)\right),
$$
$M_1$, $M_2$~--- соответствующие матрицы, определяемые по~$D_1$,
$D_2$, $A_0$, $A_1$, $A_2$. Заметим, что $M_1$~--- невырожденная
матрица из~леммы~\ref{hypAssEq}, а~значит,
система~\eqref{DelayFreeSyst} регулярна, т.~е. представима
в~нормальной форме
\begin{equation}
\label{systOrdDiff} z'(\tau)=M z(\tau),
\end{equation}
в которой $\tau\in[0,h]$, $M=M_1^{-1}M_2$. Следовательно,
$z(\tau)=e^{M\tau} z_0$, $\tau\in[0,h]$.

Систему~\eqref{DelayFreeSystBound} также можно векторизовать:
$$
L_1 z(0)+L_2 z(h)=-w.
$$
Здесь $L_1$, $L_2$~--- матрицы, которые легко построить с~помощью
произведений Кронекера, $w=\vec\left(0_{n\times n} \;0_{n\times n}
\;0_{n\times n} \;W\right)$, $0_{n\times n}$~--- нулевая матрица
размерности $n\times n$. Заметим, что $z(h)=e^{M\,h}z(0)$.
Окончательно, вектор $z_0=z(0)$ является решением алгебраической
системы с~матрицей $L=L_1+L_2e^{M\,h}$:
\begin{equation}
\label{DelayFreeSystBoundVec} L\,z_0=-w.
\end{equation}

Следующий результат вытекает из~того, что между матрицей $Z(\tau)$
и вектором $z(\tau)$ есть взаимно-однозначное соответствие.

\begin{lem}\hskip -1.5mm{\bf.}
\label{BoundAlgEx} Пусть выполнено
предположение~\textup{\ref{hypAss}}. Граничная
задача~\textup{\eqref{DelayFreeSyst},~\eqref{DelayFreeSystBound}}
\mbox{имеет} столько же решений, сколько и~алгебраическая
система~\textup{\eqref{DelayFreeSystBoundVec}}.
\end{lem}

Д о к а з а т е л ь с т в о.~ Выше было показано, что каждой
обобщенной матрице Ляпунова соответствует функция
$$
Z(\tau)=\left(\vph
Z_1(\tau),\,Z_2(\tau),\,Z_3(\tau),\,Z_4(\tau)\right),
$$
являющаяся решением системы~\eqref{DelayFreeSyst} с~граничными
условиями~\eqref{DelayFreeSystBound}. Верно и~обратное, т.~е.
каждому решению $Z(\tau)$ граничной
задачи~\eqref{DelayFreeSyst},~\eqref{DelayFreeSystBound} отвечает
некоторая обобщенная матрица Ляпунова. И~действительно, матрица
$$
U_0(\tau)=
\begin{cases}
Z_1(\tau-h),& \tau\in(h,\,2h],\\
Z_2(\tau),& \tau\in(0,\,h],\\
Z_3(\tau+h),& \tau\in(-h,\,0],\\
Z_4(\tau+2h),& \tau\in[-2h,\,-h],
\end{cases}
$$
если ее, как было описано в~начале п.~5, продолжить на~всю
вещественную прямую, является обобщенной матрицей Ляпунова. Таким
образом, доказана лемма. \hfill\square

\begin{lem}\hskip -1.5mm{\bf.}
\label{BoundLyapEx} Пусть выполнено
предположение~\textup{\ref{hypAss}} и~задана некоторая
симметричная $W$. Обобщенная матрица Ляпунова, ассоциированная
с~матрицей $W$, су\-щест\-вует и~единственна тогда и~только тогда,
когда есть единственное решение граничной
задачи~\textup{\eqref{DelayFreeSyst},~\eqref{DelayFreeSystBound}},
или, что эквивалентно, когда матрица $L$
системы~\textup{\eqref{DelayFreeSystBoundVec}} невырождена.
\end{lem}


\textbf{6. Случай $W=0$.} Отдельно стоит остановиться на~свойствах
обобщенной матрицы Ляпунова, ассоциированной с~нулевой $W$. Этот
случай не~имеет практического значения, но важен для
доказательства критерия существования и~единственности матрицы
Ляпунова.

\begin{lem}\hskip -1.5mm{\bf.}
Пусть выполнено предположение~\textup{\ref{hypAss}} и~$W=0$.
Обобщенная матрица Ляпунова~--- бесконечно дифференцируемая при
всех $\tau\in\R$ функция.
\end{lem}
\begin{proof}
Эта лемма является следствием теоремы~\ref{thmAnalyt}.
К~доказательству теоремы следует добавить равенство
$$
\Delta U_0'(0)+\Delta U_0'(-h)D_1+\Delta U_0'(-2h)D_2=0.
$$
Тогда система уравнений~\eqref{DeltaEq1}--\eqref{DeltaEq4} будет
иметь место не~только для $\tau\in(0,h)$, но и~для $\tau=0$.
Остальные рассуждения остаются без изменений. Это для производной
первого порядка. Непрерывность второй производной будет доказана,
если покажем, что
$$
\Delta U_0''(0)+\Delta U_0''(-h)D_1+\Delta U_0''(-2h)D_2=0,
$$
где $\Delta U_0''(\tau)=U_0''(\tau+0)-U_0''(\tau-0)$.
И~действительно, в~данном случае для второй производной можно
построить систему, аналогичную~\eqref{DeltaEq1}--\eqref{DeltaEq4},
используя свойства~\eqref{svD0} и~\eqref{svDneg0},
продифференцированные по~$\tau$. Эти же свойства, а~также
непрерывность первой производной матрицы Ляпунова помогут нам
установить, что
\begin{gather*}
\Delta U_0''(0)+\Delta U_0''(-h)D_1+\Delta U_0''(-2h)D_2= \\
=\Sum_{j=0}^2 U_0''(+0-j h)D_j -\Sum_{j=0}^2 U_0''(-0-j h)D_j =\Sum_{j=0}^2 U_0'(-j h)A_j ~+\\
+\Sum_{j=0}^2\Sum_{k=1}^2 D_k^T U_0''(-0+k h-j h)D_j +\Sum_{j=0}^2\Sum_{k=0}^2 A_k^T U_0'(k h-j h)D_j=\\
=\Sum_{j=0}^2\Sum_{k=0}^2 D_k^T U_0'(k h-j h)A_j +\Sum_{j=0}^2\Sum_{k=0}^2 A_k^T U_0'(k h-j h)D_j= \\
=-\Sum_{j=0}^2\Sum_{k=0}^2 A_k^T U_0(k h-j h)A_j
+\Sum_{j=0}^2\Sum_{k=0}^2 A_k^T U_0'(k h-j h)A_j=0.
\end{gather*}

Доказательство непрерывности производных высших порядков
проводится абсолютно аналогично.
\end{proof}

Для доказательства следующего технического результата будем
использовать метод, примененный в~статье~[6] для систем
запаздывающего типа.

\begin{lem}\hskip -1.5mm{\bf.}
\label{SuperDyn} Если $W=0$, то
\begin{gather}
U_0'(\tau)+D_1^TU_0'(\tau+h)+D_2^TU_0'(\tau+2h)=\notag\\
= -A_0^TU_0(\tau)-A_1^TU_0(\tau+h)-A_2^TU_0(\tau+2h),\quad
\tau\in\R. \label{svDgenTrans}
\end{gather}
\end{lem}
\begin{proof}
Введем бесконечно дифференцируемую функцию
\begin{gather*}
G(\tau)=U_0'(\tau)+D_1^TU_0'(\tau+h)+D_2^TU_0'(\tau+2h) ~+\\
+~A_0^TU_0(\tau)+A_1^TU_0(\tau+h)+A_2^TU_0(\tau+2h),\quad
\tau\in\R,
\end{gather*}
и докажем, что она тождественно равна нулю.
Из~уравнения~\eqref{svDneg0} вытекает, что $G(\tau)=0$
на~интервале $(-\infty,\,0)$. Следовательно, $G(\tau)=0$ при
$\tau\<0$ (так как матрица Ляпунова и~ее производные непрерывны).
Покажем теперь, что функция $G(\tau)$ удовлетворяет системе
\begin{gather}
\frac{d}{d\tau}\left[G(\tau)+G(\tau-h)D_1+G(\tau-2h)D_2\right]= \notag\\
=G(\tau)A_0+G(\tau-h)A_1+G(\tau-2h)A_2,\quad \tau>0.
\label{systDelVS}
\end{gather}
Это легко сделать при помощи свойств матрицы $U_0(\tau)$.
Динамическое свойство приводит к~равенству
\begin{gather*}
G(\tau)+G(\tau-h)D_1+G(\tau-2h)D_2=\\
=\sum_{j=0}^2 \sum_{k=0}^2 D_j^TU_0'(\tau+jh-kh)D_k +\sum_{j=0}^2 \sum_{k=0}^2 A_j^TU_0(\tau+jh-kh)D_k= \\
=\sum_{j=0}^2 \sum_{k=0}^2 D_j^TU_0(\tau+jh-kh)A_k +\sum_{j=0}^2
\sum_{k=0}^2 A_j^TU_0(\tau+jh-kh)D_k, \quad \tau>0.
\end{gather*}
Подставив его в~\eqref{systDelVS} и~снова применив динамическое
свойство, получим тождественное равенство.

Итак, $G(\tau)$ является решением системы
с~запаздыванием~\eqref{systDelVS}, соответствующим нулевой
начальной функции. Такое решение единственно (см.~[14]), и~это
$G(\tau)=0$, $\tau\in\R$.
\end{proof}

\begin{cor}\hskip -1.5mm{\bf.}
Если $W=0$, то
\begin{gather}
U_0'(\tau)+U_0'(\tau-h) D_1+U_0'(\tau-2h) D_2= \notag\\
= U_0(\tau) A_0+U_0(\tau-h) A_1+U_0(\tau-2h) A_2,\quad \tau\in\R,
\label{svDgen}
\end{gather}
т.~е. динамическое свойство выполнено не~только на~положительной
полуоси, но и~на~отрицательной.
\end{cor}
\begin{proof}
Введем функцию
\begin{gather*}
G(\tau)=U_0'^T(-\tau)+D_1^TU_0'^T(-\tau+h)+D_2^TU_0'^T(-\tau+2h) ~-\\
-~A_0^TU_0^T(-\tau)-A_1^TU_0^T(-\tau+h)-A_2^TU_0^T(-\tau+2h),\quad
\tau\in\R.
\end{gather*}
Остается применить к~ней рассуждения из~доказательства
леммы~\ref{SuperDyn}.
\end{proof}

Теперь докажем важное свойство обобщенных матриц Ляпунова.

\begin{lem}\hskip -1.5mm{\bf.}
\label{AnalytLyap} Пусть выполнено
предположение~\textup{\ref{hypAss}} и~$W=0$. Обобщенная матрица
Ляпунова $U_0(\tau)$ аналитична на~всей вещественной прямой $\R$.
\end{lem}
\begin{proof}
По доказанным выше свойствам, если $W=0$,
система~\eqref{DelayFreeSyst} имеет место не~только для
$\tau\in[0,h]$, но и~для всех $\tau\in\R$. Следовательно, вектор
$$
u(\tau)=\vec\left(\vph
U_0(\tau+h)\;U_0(\tau)\;U_0(\tau-h)\;U_0(\tau-2h)\right)
$$
удовлетворяет системе~\eqref{systOrdDiff} при всех $\tau\in\R$.
Отсюда очевидно, что обобщенная матрица Ляпунова имеет вид
\begin{equation}
\label{SeriesU} U_0(\tau)=\sum_{k=1}^N e^{s_k\tau} P_k(\tau),\quad
\tau\in\R,
\end{equation}
где $s_k$~--- различные собственные числа
системы~\eqref{systOrdDiff}; $P_k(\tau)$~--- некоторые матричные
полиномы.
\end{proof}

\begin{rem}
{\it Представленная в~доказательстве матрица $U_0(\tau)$
комплекснозначна, но очевидно, что ее вещественная и~мнимая части,
будучи уже вещественнозначными, также являются обобщенными
матрицами Ляпунова}.
\end{rem}
\newpage

\textbf{7. ~Критерий ~существования ~и~~единственности ~матрицы~
Ляпунова.}
\begin{thm}\hskip -1.5mm{\bf.}
Пусть выполнено предположение~\textup{\ref{hypAss}} и~задана
некоторая симметричная $W$. Для системы~\textup{\eqref{systz}}
существует единственная матрица Ляпунова, ассо\-ции\-ро\-ван\-ная
c матрицей $W$, тогда и~только тогда, когда выполнено условие
Ляпунова.
\end{thm}
\begin{proof}
Докажем \textit{необходимость}. Для заданной $W$ построим
единственную матрицу Ляпунова $U(\tau)$. Предположим, что условие
Ляпунова нарушается. По~лемме~\ref{LyapCond} это значит, что
найдется собственное число $s_0$ системы~\eqref{systz} такое, что
собственным будет и~$-s_0$. Следовательно, существуют ненулевые
векторы $p$ и~$q$, для которых
\begin{gather*}
p^T\left(s_0I+\sum_{j=1}^2 s_0 e^{-jhs_0}D_j-\sum_{j=0}^2 e^{-jhs_0} A_j\right)=0,\\
q^T\left(-s_0I-\sum_{j=1}^2 s_0 e^{jhs_0}D_j-\sum_{j=0}^2
e^{jhs_0} A_j\right)=0.
\end{gather*}
Легко убедиться непосредственной подстановкой, что найдется еще
одна матрица Ляпунова, ассоциированная с~$W$:
$
\widetilde{U}(\tau)=U(\tau)+e^{s_0\tau}qp^T
+e^{-s_0\tau}pq^T\not\equiv U(\tau),
$
что противоречит предположению.

Докажем теперь \textit{достаточность}. Пусть условие Ляпунова
выполнено, но для заданной $W$ либо не~существует матрицы
Ляпунова, либо их несколько. Это означает, что обобщенных матриц
Ляпунова либо нет, либо их несколько. По~лемме~\ref{BoundLyapEx}
матрица $L$ системы~\eqref{DelayFreeSystBoundVec} вырождена.

Вырожденность матрицы $L$ означает, что существует ненулевая
обобщенная матрица Ляпунова, ассоциированная с~$W=0$.
По~лемме~\ref{AnalytLyap} она аналитична и~имеет
вид~\eqref{SeriesU}, где по~крайней мере один из~полиномов
$P_k(\tau)$ ненулевой. Подставляем эту матрицу
в~равенство~\eqref{svDgen}, которое должно обратиться в~тождество
\begin{gather}
\sum_{k=1}^N s_ke^{s_k\tau} P_k(\tau)+\sum_{k=1}^N e^{s_k\tau} P_k'(\tau)~+ \notag\\
+~\sum_{k=1}^N s_ke^{s_k(\tau-h)} P_k(\tau-h)D_1 +\sum_{k=1}^N e^{s_k(\tau-h)} P_k'(\tau-h)D_1 ~+ \notag\\
+~\sum_{k=1}^N s_ke^{s_k(\tau-2h)} P_k(\tau-2h)D_2 +\sum_{k=1}^N e^{s_k(\tau-2h)} P_k'(\tau-2h)D_2= \notag\\
= \sum_{k=1}^N e^{s_k\tau} P_k(\tau) A_0+\sum_{k=1}^N
e^{s_k(\tau-h)} P_k(\tau-h) A_1+\sum_{k=1}^N e^{s_k(\tau-2h)}
P_k(\tau-2h) A_2. \label{lastThVs}
\end{gather}
Будем считать, что полином $P_1(\tau)$ ненулевой, его степень
обозначим числом $\ell$, а~коэффициент при старшей степени через
$P_1^{(\ell)}$. В~тождестве~\eqref{lastThVs} приравняем
коэффициенты при $e^{s_1\tau} \tau^{\ell}$ в~правой и~левой
частях. Получаем равенство
$$
P_1^{(\ell)}\left(s_1I+s_1e^{-s_1 h}D_1+s_1e^{-2s_1
h}D_2-A_0-e^{-s_1 h}A_1-e^{-2s_1 h}A_2\right)=0.
$$
А раз $P_1^{(\ell)}\neq 0$, то $s_1$ есть собственное число
системы~\eqref{systz}.

Теперь проделаем те же операции с~уравнением~\eqref{svDgenTrans}.
Итог~--- $$ \left(s_1I+s_1e^{s_1 h}D_1^T+s_1e^{2s_1
h}D_2^T+A_0^T+e^{s_1 h}A_1^T+e^{2s_1 h}A_2^T\right)P_1^{(\ell)}=0.
$$
Оказывается, что $-s_1$ тоже является собственным числом
системы~\eqref{systz}. Таким образом, доказано, что условие
Ляпунова нарушается.
\end{proof}

Теперь стал очевиден результат, о~котором мы говорили, когда ввели
определение обобщенной матрицы Ляпунова.

\begin{cor}\hskip -1.5mm{\bf.}
Если выполнены предположение~\textup{\ref{hypAss}} и~условие
Ляпунова, определения~\textup{\ref{MatrL}}
и~\textup{\ref{MatrLgen}} эквивалентны.
\end{cor}


\textbf{8. Заключение.} В~статье исследуются свойства матриц
Ляпунова для линейных стационарных систем с~запаздыванием
нейтрального типа. Сделан первый шаг в~решении вопроса
существования и~единственности матрицы Ляпунова. Главная проблема,
которую предстоит решить,~--- как отказаться
от~предположения~\ref{hypAss}, ограничивающего класс
рассматриваемых систем.
