{\footnotesize

\vskip2.5mm

\begin{list}{}{\leftmargin=7mm \rightmargin=7mm \listparindent=5mm}

\item \hskip5mm Earlier matrices with multiple eigenvalues were
considered only for theoretical purposes. However, now such
non-generic matrices are also of practical interest because they
appear in different problems of quantum mechanics, nuclear
physics, optics and dynamic of mechanical systems. A square matrix
with elements that are linearly dependent on a parameter is
considered in this paper. A method to find the values of the
parameter such that the matrix has a repeated eigenvalue is
considered. We find the polynomial whose roots are these values of
the parameter using only finite number of algebraic operations on
the matrix elements. The method can be generalized to matrices
with elements algebraically dependent on a parameter. A numerical
example of how to find the required parameter values is
considered. Refs~16.

\textit{Keywords}: the Kronecker product, the Leverrier method,
the Newton sums.

\end{list}

{\bf Постановка задачи и~известные результаты.} Даны две
квадратные ($k\times k$) матрицы $A$ и~$B$ с~комплексными
элементами. Требуется определить все значения $\lambda$, при
которых матрица $A+\lambda B$ имеет кратные собственные числа.

З а~м е ч а~н и~е~ 1. В~дальнейшем будем предполагать, что
у~матриц $A$ и~$B$ нет общих собственных чисел. Проверить это
и~найти все общие собственные значения матриц можно с~помощью
алгоритма, предложенного в~работе~[1].

Существование кратных собственных чисел у~матрицы не~является
общим слу\-чаем. Ранее исследование таких вырожденных матриц было
в~основном теоретичес\-ким. Однако решение данной задачи сейчас
находит и~практические приложения в~квантовой механике, ядерной
физике, оптике, динамике механических систем~[2,~3].

Подобные задачи рассматриваются в~теории возмущений. Есть довольно
много работ, посвященных числам обусловленности матриц с~кратными
собственными числами (см., например, статьи последних лет [4--6]).
Однако в~теории возмущений решаются задачи, в~которых величины
$\lambda$ близки к~нулю, а~мы будем считать, что $\lambda$
принимает произвольные значения.

Подробное описание такой задачи, ее свойства и~приложения можно
найти в~работе~[7]. В~ней приводится и~решение, основанное
на~методе Ньютона. Но не~существует глобально сходящихся численных
методов, позволяющих найти все требуемые значения $\lambda$.
В~связи с~этим задача не~теряет своей актуальности, и~в~недавней
статье~[8] предлагается метод, который дает возможность определить
все величины $\lambda$, применяя только стандартные вычислительные
средства. К~сожалению, он весьма чувствителен к~погрешностям, как
замечают и~сами авторы статьи, потому его можно использовать
только для матриц малых порядков.

В настоящей работе предлагается метод, с~помощью которого
за~конечное число алгебраических операций над элементами матриц
$A$ и~$B$ строится полином, корни которого являются требуемыми
значениями $\lambda$. Найти корни этого полинома можно с~любой
необходимой точностью. Данный метод использует свойства
кронекеровского произведения матриц и~метод Леверье построения
характеристического полинома~[9]. Приводится пример применения
этого метода, и~результаты сравниваются с~полученными
в~статье~[8].

{\bf Предварительные результаты.} Рассмотрим полином
$$f(x)=a_0x^n+a_1x^{n-1}+a_2x^{n-2}+\ldots+a_n,\qquad
a_j\in\mathbb{C},~~j=0,1,2,\ldots,n.$$ Пусть
$\lambda_1,\lambda_2,\ldots,\lambda_n\in\mathbb{C}$~--- его корни
(среди них могут быть совпадающие).

{\bf Определение 1.} \emph{Суммами Ньютона полинома} $f(x)$
\emph{называются величины}
$$s_p=\lambda_1^p+\lambda_2^p+\ldots+\lambda_n^p,\qquad p=0,1,2,\ldots\ .$$

Известна формула~[10], выражающая коэффициенты полинома $f(x)$
через его суммы Ньютона (считаем, что $a_0=1$):
\begin{equation}\label{a_p}
a_p=\frac{1}{p!}\left|\begin{array}{cccccc}
s_1&1&0&0&\ldots&0\\
s_2&s_1&2&0&\ldots&0\\
s_3&s_2&s_1&3&\ldots&0\\
\ldots&&&&&\\
s_p&s_{p-1}&s_{p-2}&s_{p-3}&\ldots&s_1
\end{array}\right|_{p\times p}.
\end{equation}

Cуммы Ньютона характеристического полинома матрицы $D_{k\times k}$
выражаются через степени этой матрицы по~формуле~[9]
%\begin{equation}\label{trace}
$$s_p={\rm Sp}\, D^p,\qquad p=1,2,\ldots,$$
%\end{equation}
где ${\rm Sp}\, D$ обозначает след матрицы $D$.

{\bf Определение 2.} \emph{Кронекеровским произведением матриц}
$A_{k\times k}$ и~$B_{l\times l}$ \emph{назы\-вается матрица}
$$[A\otimes
B]_{kl\times kl}=\left[\begin{array}{cccc}
a_{11}B&a_{12}B&\ldots&a_{1k}B\\
a_{21}B&a_{22}B&\ldots&a_{2k}B\\
\ldots&&&\\
a_{k1}B&a_{k2}B&\ldots&a_{kk}B \end{array}\right] .$$

В дальнейшем нам понадобится следующее свойство кронекеровского
произведения матриц~[11]:

\begin{equation}\label{prop3}
(A\otimes B)(C\otimes D)=(AC)\otimes(BD).
\end{equation}

Обозначим через ${\mathcal C}_D$ матрицу
%\begin{equation}\label{C}
$${\mathcal C}_D=E\otimes D-D\otimes E.$$
%\end{equation}
Здесь $E$~--- единичная матрица того же порядка, что и~матрица $D$
(размерности $k\times k$).

Пусть $\lambda_1,\lambda_2,\ldots,\lambda_k$~--- собственные числа
матрицы $D$ (среди них могут быть совпадающие). Справедлива
следующая теорема~[12]:

\textbf{Теорема 1.} {\it Собственные числа матрицы ${\mathcal
C}_D$ равны $\lambda_i-\lambda_j$, где $i,j=1,2,\ldots,k$.}

{\bf Следствие 1.} {\it Матрица $D$ имеет кратные собственные
числа тогда и~только тогда, когда кратность собственного числа $0$
больше, чем $k$. Матрица $D$ не~имеет кратных собственных чисел
тогда и~только тогда, когда кратность ее собственного числа $0$
равна $k$}.


{\bf Кратные собственные числа матрицы.} Обозначим через $s_p$
и~$S_p$ ($p=0,1,2,\ldots$) суммы Ньютона характеристических
полиномов матриц $D$ и~${\mathcal C}_D$ соответственно.
Справедлива следующая теорема, показывающая связь между суммами
Ньютона $s_p$ и~$S_p$ (напомним, что $s_p={\rm Sp}\, D^p,S_p={\rm
Sp}\, {\mathcal C}_D^p$):

{\bf Теорема 2.} {\it След матрицы ${\mathcal C}_D^p$ находится
по~формулам}
\begin{eqnarray}\label{Sp}
S_{2p}&=&2ks_{2p}-2{C}_{2p}^1s_{2p-1}s_1+2{C}_{2p}^2s_{2p-2}s_2-\ldots+(-1)^P{C}_{2p}^ps_p^2,\\
S_{2p-1}&=&0,\nonumber \end{eqnarray} где $\displaystyle
{C}_n^p=\frac{n!}{p!(n-p)!}$.

Д о~к~а з а~т е л ь с~т в~о. Воспользовавшись
формулой~(\ref{prop3}), находим, что
$${\mathcal C}_D^p=D^p\otimes E-{C}_p^1D^{p-1}\otimes D+{C}_p^2D^{p-2}\otimes D^2-\ldots+(-1)^{p}E\otimes D^p.$$
Учитывая свойства следа кронекеровского произведения матриц~[11],
сразу получаем требуемое.

{\bf Теорема 3.} {\it Матрица $D$ имеет кратные собственные числа
тогда и~только тогда, когда}
\begin{equation}\label{pol}
\left|\begin{array}{ccccc}
S_2&2&0&\ldots&0\\
S_4&S_2&4&\ldots&0\\
\ldots&&&&\\
S_{k^2-k}&S_{k^2-k-2}&S_{k^2-k-4}&\ldots&S_2
\end{array}\right|_{(k^2-k)/2\times (k^2-k)/2}=0.
\end{equation}

Д о~к~а з а~т е л ь с~т в~о. С~помощью формулы~(\ref{a_p}) выразим
коэффициент $a_{k^2-k}$ при $\mu^k$ характеристического полинома
матрицы ${\mathcal C}_D$
$$\det({\mathcal C}_D-\mu E_{k^2\times k^2})=a_0\mu^{k^2}+a_1\mu^{k^2-1}+a_2\mu^{k^2-2}+\ldots+a_{k^2}$$
через его суммы Ньютона $S_p={\rm Sp}\, {\mathcal C}_D^p$:
$$a_{k^2-k}=\frac{1}{(k^2-k)!}\left|\begin{array}{ccccccc}
0&1&0&0&\ldots&0&0\\
S_2&0&2&0&\ldots&0&0\\
0&S_2&0&3&\ldots&0&0\\
\ldots&&&&&&\\
0&S_{k^2-k-2}&0&S_{k^2-k-4}&\ldots&0&k^2-k+1\\
S_{k^2-k}&0&S_{k^2-k-2}&0&\ldots&S_2&0
\end{array}\right|.$$
По теореме Лапласа~[13] разложим данный определитель, выбрав все
столбцы с~четными номерами:
$$a_{k^2-k}=
\frac{1}{(k^2-k)!}\left|\begin{array}{ccccc}
1&0&0&\ldots&0\\
S_2&3&0&\ldots&0\\
S_4&S_2&5&\ldots&0\\
\ldots&&&&\\
S_{k^2-k-2}&S_{k^2-k-4}&S_{k^2-k-6}&\ldots&k^2-k-1
\end{array}\right|\times$$
$$\times\left|\begin{array}{ccccc}
S_2&2&0&\ldots&0\\
S_4&S_2&4&\ldots&0\\
\ldots&&&&\\
S_{k^2-k}&S_{k^2-k-2}&S_{k^2-k-4}&\ldots&S_2
\end{array}\right|=$$
$$=\frac{1}{(k^2-k)!!}\left|\begin{array}{ccccc}
S_2&2&0&\ldots&0\\
S_4&S_2&4&\ldots&0\\
\ldots&&&&\\
S_{k^2-k}&S_{k^2-k-2}&S_{k^2-k-4}&\ldots&S_2
\end{array}\right|,
$$
где $(k^2-k)!!$ обозначает произведение всех четных натуральных
чисел от~1 до~$k^2-k$ включительно. Из~последнего равенства сразу
следует утверждение теоремы.

Теоремы 1--3 дают возможность построить алгоритм нахождения
значений $\lambda$, для которых матрица $D=A+\lambda B$ имеет
кратные собственные числа. Требуется вычислить следы степеней
данной матрицы. Для этого воспользуемся свойствами следа
матрицы~[14].

{\bf Лемма.} ${\rm Sp}\,(AB)={\rm Sp}\, (BA)$, ${\rm Sp}\,
(A+B)={\rm Sp}\, A+{\rm Sp}\, B$.

Имеем следующее равенство:
\begin{equation}\label{SpD}
s_p={\rm Sp}\, D^p={\rm Sp}\, A^p+{\sf C}_p^1({\rm Sp}\,
A^{p-1}B)+{\sf C}_p^2{\rm Sp}\, (A^{p-2}B^2)+\ldots+{\rm Sp}\,
B^p\ (p=1,2,3,\ldots).
\end{equation}

Таким образом, нужно определить степени матриц $A$ и~$B$ от~первой
до~$k^2-k$-й включительно.

З а~м е ч а~н и~е\, 2. Последнюю операцию матричного умножения при
вычислении следа можно не~выполнять, достаточно установить лишь
элементы, стоящие на~главной диагонали произведения.

З а~м е ч а~н и~е~ 3. Расчет степени матрицы~--- задача довольно
трудоемкая. Для матриц большого порядка наиболее быстро это можно
сделать с~помощью метода Штрассена~[15].

{\bf Алгоритм.} Пусть имеются две квадратные матрицы $A_{k\times
k}$ и~$B_{k\times k}$. Требуется найти все значения $\lambda$
такие, что матрица $A+\lambda B$ имеет кратные собственные числа.

\textbf{1.} Вычисляем степени матриц $A$ и~$B$: $A^p,B^p\quad
(p=1,2,3,\ldots,k^2-k-1)$.

\textbf{2.} Получаем следы матриц $A^pB^q$
$p,q\in\{0,1,2,\ldots,k^2-k\},~~p+q\le k^2-k$.

\textbf{3.} По~формуле~(\ref{SpD}) рассчитываем суммы Ньютона
$s_p$ характеристического полинома матрицы $D=A+\lambda B$
$(p=1,2,\ldots,k^2-k)$.

\textbf{4.} По~формуле~(\ref{Sp}) находим суммы Ньютона
$S_{2p}={\rm Sp}\, D^{2p}$ характеристического полинома матрицы
${\mathcal C}_D$ $(p=1,2,\ldots,(k^2-k)/2)$.

\textbf{5.} Вычисляем корни полинома~(\ref{pol}).

Эти корни и~есть требуемые значения $\lambda$.

З а~м е ч а~н и~е~ 4. Метод допускает обобщение на~случай матрицы
$D(\lambda)$, элементы которой являются полиномами от~$\lambda$
степени выше первой, т.~е. матричного полинома
$$D(\lambda)=A_0\lambda^m+A_1\lambda^{m-1}+\ldots+A_m,$$
где $A_j$, $j=0,1,\ldots,m,$~--- квадратная матрица $k$-го порядка
с комплексными элементами. В~данном случае алгоритм усложняется
при нахождении сумм Ньютона характеристического полинома матрицы
$D(\lambda)$. Все остальные его шаги не~изменяются. (Обзор
различных задач, связанных с~собственными числами матрицы,
и~методов их решения см. в~книге Дж. Х. Уилкинсона~[16].)

{\bf Численный пример.} Рассмотрим задачу из~статьи~[8]. Даны
матрицы
$$A=\left(\begin{array}{rrr}
1&-2&3\\
-1&1&2\\
1&1&-1 \end{array}\right),\qquad B=\left(\begin{array}{rrr}
1&-1&1\\
1&1&3\\
-1&1&1 \end{array}\right).$$ Требуется найти все значения
параметра $\lambda$, при которых матрица $D=A+\lambda B$ имеет
кратные корни.

Cуммы Ньютона характеристического полинома матрицы $A+\lambda B$
\begin{eqnarray*}
s_1&=&3\lambda+1,\\
s_2&=&5\lambda^2+6\lambda+17,\\
s_3&=&21\lambda^3+33\lambda^2+30\lambda-8,\\
s_4&=&65\lambda^4+124\lambda^3+82\lambda^2+4\lambda+117,\\
s_5&=&173\lambda^5+415\lambda^4+485\lambda^3+285\lambda^2+240\lambda-134,\\
s_6&=&473\lambda^6+1386\lambda^5+2121\lambda^4+1620\lambda^3+522\lambda^2-324\lambda+890.
\end{eqnarray*}

Cуммы Ньютона характеристического полинома матрицы ${\mathcal
C}_A$
\begin{eqnarray*}
S_2&=&12\lambda^2+24\lambda+100,\\
S_4&=&186\lambda^4+504\lambda^3+1980\lambda^2+2424\lambda+4234,\\
S_6&=&-11280\lambda^6-33840\lambda^5-35904\lambda^4+6480\lambda^3+36282\lambda^2+42300\lambda+64058.
\end{eqnarray*}

По теореме 3 матрица $D=A+\lambda B$ имеет кратные корни тогда
и~только тогда, когда
$$\left|\begin{array}{ccc}
S_2&2&0\\
S_4&S_2&4\\
S_6&S_4&S_2
\end{array}\right|=$$
$$=-101\ 904\lambda^6-323\ 424\lambda^5-550\ 032\lambda^4-523\ 584\lambda^3-1018\ 848\lambda^2-1005\ 696\lambda-1027\ 936=0.$$
Корни этого уравнения равны
$$-2.333069484; -1.401818975\pm 0.6190045476i, 0.2836993683\pm 0.1543575855i, 1.933794680.$$
Данные значения $\lambda$ отличаются от~приведенных в~работе~[8]
уже в~первом знаке после точки:
$$1.5628;-2.2078;-1.1690\pm 0.8436i;0.2735\pm 0.0988i .$$
Сравним собственные числа матрицы $A+\lambda B$ при
$\lambda=-2.333069484$ и~$\lambda=-2.2078$.

Для $\lambda=-2.333069484$ получаем собственные числа
$$-0.257071186441648280;-0.257116318877424144;-5.48502094668092610.$$
Видно, что первые два собственных числа совпадают до~тысячных.

Для $\lambda=-2.2078$ получаем собственные числа
$$0.567197754532772214;-1.01664058066736440;-5.17395717386540620.$$
Здесь все собственные числа различны.

Как справедливо замечают авторы работы~[8], их метод дает довольно
большую погрешность. В~связи с~этим они предлагают применять его
лишь для матриц малых порядков. Однако, по~нашему мнению, даже
в~этом случае погрешность довольно велика.

{\bf Заключение.} В~работе предлагается метод, позволяющий
построить алгебраическое уравнение, корнями которого являются
значения параметра $\lambda$, которым соответствуют кратные
собственные числа матрицы $D=A+\lambda B$. В~отличие от~метода,
изложенного ранее в~[8], он более точный, так как не~требуется
находить сингулярные разложения матриц. Коэффициенты
результирующего уравнения получаются путем выполнения конечного
числа алгебраических операций над элементами исходных матриц
и~могут быть вычислены с~произвольной точностью. Данный метод
может быть обобщен на~случай матрицы $D(\lambda)$, элементы
которой являются полиномами от~$\lambda$ степени выше первой.
