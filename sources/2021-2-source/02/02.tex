{\leftskip=7mm\noindentРассматривается задача изгиба прямоугольной пластины, подкрепленной перекрестной системой ребер жесткости.
На пластину, кроме поперечной нагрузки, действуют передаваемые через ребра силы в~ее плоскости. Получено аналитическое решение граничной задачи для разрешающего дифференциального уравнения относительно
нормального прогиба пластины, которое описывает деформацию прямоугольной пластины, подкрепленной ребрами жесткости. Решение представлено в~виде рядов по комбинациям регулярных и~специальных разрывных функций, которые быстро сходятся, и~приводит к~простому вычислительному алгоритму.
Влияние ребер учитывается в~уравнении в~виде дополнительных слагаемых, содержащих множители с~дельта-функцией. Такой подход дает возможность освободиться от ряда предположений, касающихся взаимодействия пластины с~подкрепляющими ее элементами. Использование аппарата обобщенных функций при моделировании объектов указанного типа упрощает граничные условия
(отсутствуют условия сопряжения различных элементов конструкции), но при этом усложняются дифференциальные уравнения: задача сводится к~так называемым частично вырожденным уравнениям. Разработка аналитических методов, позволяющих получать точные решения дифференциальных уравнений такого типа, и~внедрение их в~расчeтную практику являются одной из актуальных задач механики объектов с~нарушенной регулярностью.\\[1mm]
\textit{Ключевые слова}: пластина, ребра жесткости, математическая модель, численно-аналитические методы, специальные разрывные функции, функция Дирака, функция Хэвисайда, ряды Фурье, ортогональные ряды.

\textbf{1. Введение.} Настоящая работа посвящена развитию численно-аналитического
метода расчета упругой тонкостенной конструкции
нерегулярной структуры --- сложной составной конструкции, состоящей из обшивки (тонкой пластины) и~стержневой системы
(ребер жесткости). Особенность конструкции состоит в~том, что ребра жесткости являются не вспомогательными,
а основными несущими элементами наряду с~обшивкой (пластиной): они имеют значительную жесткость и~соответствующие размеры.


В расчетах подобных конструкций изначально использовался способ расчленения их на
отдельные элементы с~плавно меняющейся геометрией
и жесткостью, с~последующей «склейкой» решений для этих элементов. Такой подход
приводит к~громоздкому, а~иногда и~трудно
реализуемому алгоритму, применение которого зачастую затруднено неустойчивостью счета,
связанной с~различной изменяемостью
искомых функций.


Современные ЭВМ дают возможность широкого внедрения численных методов определения
напряжений и~деформаций в~элементах конструкций сложной формы. Среди таких методов наибольшее практическое
значение имеет метод конечных элементов (МКЭ).


С позиций МКЭ конструкция представляет собой композицию различных конечных элементов:
 оболочек нулевой кривизны (пластинок),
моделирующих обшивку и~стенки ребер жесткости --- ригелей и~ диафрагм; дискретных
стержневых элементов, имеющих продольную,
крутильную и~изгибную жесткости и~моделирующих поя\-са ригелей; диафрагм и~т.~п.
Взаимодействие конструкции с~опорами моделируется введением специальной системы связей.

Продолжаются исследования сложной составной конструкции, состоящей из пластины-обшивки и~стержневой
системы --- ребер жесткости. В~работах [1, 2] получены
аналитические решения краевых задач для уравнений деформирования прямоугольной
пластины, подкрепленной ребрами жесткости.
Расчетные модели конструкций реализованы в~аналитических решениях, которые строятся с~помощью методов Канторовича и~Стеклова\,---\,Лиувилля\,---\,Фубини.
Решение представлено как ряды по комбинациям регулярных и~специальных разрывных
функций, которые быстро сходятся, и~приводит к~простому вычислительному алгоритму.
В статье [3] на основе генетического алгоритма и~теории сдвиговых
деформаций первого порядка предложен
метод оптимизации положения ребер жесткой пластины, опирающейся на упругое основание.
В работе [4] функция прогиба ребристой прямоугольной пластины опреде\-ляется методом
Канторовича\,---\,Власова в~виде отрезка ряда
$\sum_{i = 1}^n {{W_i}\left( y \right){X_i}\left( x \right)}$ с~функциями
${X_i}\left( x \right)$,
удовлетворяющими заданным кинематическим граничным условиям, а~функции
${W_i}\left( y \right)$
находятся как решения дифференциальных уравнений четвертого порядка.
В первом приближении предлагается одночленная аппроксимация с~использованием
специальной балочной функции ${X_1}\left( x \right)$,
определяемой методом начальных параметров. В~[5] рассматривается метод решения
проблемы свободных колебаний косоребристых пластин.
В работе [6], применительно к~бетону или композитным железобетонным конструкциям,
разработана модель для анализа пластин,
усиленных параллельными балками, включая эффекты ползучести и~усадки.
Экспериментальным исследованиям характеристик изгиба
предварительно напряженных ребристых плит посвящена статья [7].
В работе [8] предлагается метод наименьших квадратов для анализа изгиба прямоугольных
ребристых пластин.
Ребристая пластина прямоугольной формы рассматривается как составная конструкция
пластина---ребра.
Используется условие совместности перемещений пластины и~ребер.
Эффект деформации сдвига в~пластинах, усиленных параллельными балками, анализируется в~[9].
Анализ пластин, усиленных параллельными балками, проводится и~в~работе [10].
Модель для анализа пластин, армированных балками, учитывающая
результирующие плоские силы и~деформации пластины, а~также осевые силы и~деформации
балок из-за комбинированной реакции системы, исследуется в~[11].
В работах [12--14] представлено сравнение двух аналитических подходов к~анализу
деформирования пластин~--- метода аналитической декомпозиции и~метода дельта-функций.

\textbf{2. Постановка задачи и~ее решение.} Рассматривается прямоугольная пластина
$\left( {0 \le x \le a,0 \le y \le b} \right)$,
подкреплeнная ребрами, расположенными параллельно осям $x$ и~$y$ по линиям
$x = {x_j},j = 1,2,\dots,{K_2}$, и~$y = {y_i},i = 1,2,\dots,{K_1}$. Если пластина нагружена поперечной нагрузкой
$q\left( {x,y} \right)$ и~в~условиях
задачи имеются еще и~силы, действующие в~срединной плоскости пластины, то последние
могут оказать значительное
влияние на изгиб пластины, и~потому при выводе основных разрешающих уравнений их
необходимо принимать в~расчет. Не останавливаясь на подроб\-нос\-тях вывода соответствующих
уравнений (особенности можно проследить, например, в~[15--17]), отметим, что задача
сводится к~основному разрешающему уравнению относительно нормального прогиба $w$:
\[
{\nabla ^4}w = \frac{{q\left( {x,y} \right)}}{D} + \frac{{T_1^*}}{D}
\frac{{{\partial ^2}w}}{{\partial {x^2}}} +
 \frac{{T_2^*}}{D}\frac{{{\partial ^2}w}}{{\partial {y^2}}} + \frac{{2S}}{D}
\frac{{{\partial ^2}w}}{{\partial x\partial y}} \,-
\]
\[
-\, \sum\limits_{i = 1}^{{K_1}} {{\lambda _{1i}}\frac{{{\partial ^{{\rm{ 4}}}}w}}
{{\partial {x^4}}}\delta \left( {y - {y_i}} \right)}
 - \sum\limits_{j = 1}^{{K_2}} {{\lambda _{2j}}\frac{{{\partial ^{{\rm{ 4}}}}w}}
{{\partial {y^4}}}\delta \left( {x - {x_j}} \right)},
\]
где $D = \frac{{E{h^3}}}{{12\left( {1 - {\nu ^2}} \right)}}$ --- цилиндрическая жeсткость пластины;
$E$ — модуль Юнга; $\nu$~--- коэффициент Пуассона; $h$~--- толщина пластины;
${\lambda _{1i}} = \frac{{{E_{1i}}{J_{1i}}}}{D}$, ${\rm{ }}i = 1,2,\ldots,{K_1}$;
${\rm{ }}{\lambda _{2j}} = \frac{{{E_{2j}}{J_{2j}}}}{D}$, ${\rm{ }}j = 1,2,\ldots,{K_2}$;
${E_{1i}}{J_{1i}},{\rm{ }}{E_{2j}}{J_{2j}}$ --- изгибные жесткости ребер
(произведение модуля упругости Юнга на момент инерции поперечного сечения ребра), расположенных в~первом и~втором направлениях;
$T_1^*,T_2^*,S$ --- заданные силы, действующие в~плос\-кости пластины; $\delta $ --- дельта-функция Дирака.


Пусть на пластину в~ее плоскости действуют сжимающие или растягивающие силы
${P_{1i}},{\rm{ }}{P_{2j}}$, которые передаются через ребра жесткости в~обоих направлениях.
Соот\-вет\-ст\-вующие погонные усилия, действующие по кромкам $y = 0,b$ и~$x = 0,a$ будут выражаться формулами

\[T_2^* = \sum\limits_{j = 1}^{{K_2}} {{P_{2j}}\delta \left( {x - {x_j}} \right)}, \,\,{\rm{ }}T_1^* = \sum\limits_{i = 1}^{{K_1}}
 {{P_{1i}}\delta \left( {y - {y_i}} \right)}. \]
Уравнение изгиба с~учетом этих сил имеет вид

\[{\nabla ^4}w = \frac{{q\left( {x,y} \right)}}{D} -
 \sum\limits_{i = 1}^{{K_1}} {\left[ {{\lambda _{1i}}\frac{{{\partial ^{\rm{4}}}w}}{{\partial {x^4}}} +
 \frac{{{P_{1i}}}}{D}\frac{{{\partial ^{\rm{2}}}w}}{{\partial {x^2}}}} \right]\delta \left( {y - {y_i}} \right)}  \,- \]

\begin{equation}
\label{eq1}
-\, \sum\limits_{j = 1}^{{K_2}} {\left[ {{\lambda _{2j}}\frac{{{\partial ^{\rm{4}}}w}}{{\partial {y^4}}} +
 \frac{{{P_{2j}}}}{D}\frac{{{\partial ^{\rm{2}}}w}}{{\partial {y^2}}}} \right]\delta \left( {x - {x_j}} \right)}.
\end{equation}

\noindent
Граничные условия могут быть любые. Если, например, кромки $x = 0, a$  свободны, то получаем уравнения

\[ - D\left( {\frac{{{\partial ^2}w}}{{\partial {x^{{\rm{ }}2}}}} +
 \nu \frac{{{\partial ^2}w}}{{\partial {y^{{\rm{ }}2}}}}} \right) = g\left( y \right),\]
\[\left( {\frac{{{\partial ^3}w}}{{\partial {x^{{\rm{ }}3}}}} +
\left( {2 - \nu } \right)\frac{{{\partial ^3}w}}{{\partial x\partial {y^{{\rm{ }}2}}}}} \right) = 0,\]
а когда кромки $y = 0, b$  шарнирно оперты, то при $y = 0, b$ имеем, что

\[{\rm{ }} - D\frac{{{\partial ^2}w}}{{\partial {y^2}}} = f\left( x \right), \,\,w = 0.\]
Здесь функции $f\left( x \right)$ и~${\rm{ }}g\left( x \right)$ зависят от способа передачи усилий через ребра жест\-кос\-ти.
Если эти усилия передаются через срединную плоскость пластины-обшивки, то $f\left( x \right) = g\left( x \right) = 0$.
Если же, например, указанные силы передаются через центры тя\-жес\-ти поперечных сечений ребер жесткости, то изгибающие моменты
от «внецентренного» растяжения или сжатия необходимо учесть в~краевых условиях для пластины. В~последнем случае функции
$f\left( x \right)$ и~${\rm{ }}g\left( x \right)$ определяются по формулам

\[f\left( x \right) = \sum\limits_{j = 1}^{{K_2}} {{P_{2j}}{e_{2j}}\delta \left( {x - {x_j}} \right)},\,\,
{\rm{ }}g\left( y \right) = \sum\limits_{i = 1}^{{K_1}} {{P_{1i}}{e_{1i}}\delta \left( {y - {y_i}} \right)} \]
и учитывают моменты, действующие на торцах ребер жесткости, возникающие из-за эксцентричного приложения сил ${P_{1i}}$
и ${P_{2j}}$ (${e_{1i}}$ и~${e_{2j}}$ --- эксцентриситеты сил  ${P_{1i}}$ и~${P_{2j}}$).
Возможны и~другие случаи передачи усилий, когда моменты необходимо учитывать как в~граничных условиях,
так и~в~расчете ребер жесткости.

Запишем общее решение уравнения (\ref{eq1}) в~виде
\[w\left( {x,y} \right) = \sum\limits_{m = 1}^\infty  {\sum\limits_{n = 1}^\infty
  {\frac{{{q_{mn}}\sin \left( {{\alpha _m}x} \right)\sin \left( {{\beta _n}y} \right)}}
{{D{{\left( {\alpha _m^2 + \beta _n^2} \right)}^2}}}} } \, -  \]
\[ - \, \sum\limits_{n = 1}^\infty  {\sum\limits_{k = 1}^4 {{C_{kn}}{Z_{{\beta _n}k}}\left( x \right)\sin \left( {{\beta _n}y} \right)} }
   - \sum\limits_{m = 1}^\infty  {\sum\limits_{k = 1}^4 {{D_{km}}{Z_{{\alpha _m}k}}\left( y \right)\sin \left( {{\alpha _m}x} \right) - } } \]
\[ -\, \sum\limits_{m = 1}^\infty  {\sum\limits_{i = 1}^{{K_1}} {\left[ {{\lambda _{1i}}\alpha _m^4 +
 \frac{{{P_{1i}}}}{D}\alpha _m^2} \right]{w_{my}}\left( {{y_i}} \right)\Psi _{_{i{\alpha _m}}}^*\left( y \right)\sin \left( {{\alpha _m}x} \right)} } \, - \]
\begin{equation}
\label{eq2}
 - \, \sum\limits_{j = 1}^{{K_2}} {\sum\limits_{n = 1}^\infty  {\left[ {{\lambda _{2j}}\beta _n^4 +
 \frac{{{P_{2j}}}}{D}\beta _n^2} \right]{w_{nx}}\left( {{x_j}} \right)\Psi _{j{\beta _n}}^*\left( x \right)\sin \left( {{\beta _n}y} \right)} } .
\end{equation}
Здесь
\[{q_{mn}} = \frac{4}{{ab}}\int\limits_0^a {\int\limits_0^b {q\left( {x,y} \right)
\sin \left( {{\alpha _m}x} \right)\sin \left( {{\beta _n}y} \right)dxdy} }, \]
$\left\{ {{Z_{{\alpha _m}k}}\left( y \right)} \right\},\left\{ {{Z_{{\beta _n}k}}\left( x \right)} \right\}$ --- соответствующие
фундаментальные системы решений
однородных уравнений $\nabla _{{\alpha _m}}^4{Z_{{\alpha _m}k}} = 0,\,{\rm{ }}\nabla _{{\beta _n}}^4{Z_{{\beta _n}k}} = 0,$
где $\nabla _{{\alpha _m}}^4,\nabla _{{\alpha _m}}^4$ --- дифференциальные операторы
\[\nabla _{{\alpha _{\rm{m}}}}^4 = {\left( {\frac{{{d^2}}}{{d{y^2}}} - \alpha _m^{{\rm{ }}2}} \right)^2}, \,
{\rm{ }}\nabla _{{\beta _{\rm{n}}}}^4 = {\left( {\frac{{{d^2}}}{{d{x^2}}} - \beta _n^{{\rm{ }}2}} \right)^2},\]
функции $\Psi _{i{\alpha _m}}^ * \left( y \right),\Psi _{j{\beta _n}}^ * \left( x \right)$ вычисляются по формулам
\[\Psi _{j{\beta _n}}^ * \left( x \right) = {\Delta _{{\beta _n}}}\left( {x - {x_j}} \right)\theta \left( {x - {x_j}} \right), \,\,
\Psi _{i{\alpha _m}}^ * \left( y \right) = {\Delta _{{\alpha _m}}}\left( {y - {y_i}} \right)\theta \left( {y - {y_i}} \right),\]
\[{\Delta _{{\beta _n}}}\left( {x - {x_j}} \right) = \frac{1}{{2\beta _n^3}}\left\{ {{\beta _n}\left( {x - {x_j}}
 \right){\mathop{\rm ch}\nolimits} {\beta _n}\left( {x - {x_j}} \right) -
 {\mathop{\rm sh}\nolimits} {\beta _n}\left( {x - {x_j}} \right)} \right\},\]
\[{\Delta _{{\alpha _m}}}\left( {y - {y_i}} \right) = \frac{1}{{2\alpha _m^3}}\left\{ {{\alpha _m}\left( {y - {y_i}}
 \right){\mathop{\rm ch}\nolimits} {\alpha _m}\left( {y - {y_i}} \right) -
 {\mathop{\rm sh}\nolimits} {\alpha _m}\left( {y - {y_i}} \right)} \right\},\]
в которых $\theta $ --- функция единичного скачка Хэвисайда.









В решение (\ref{eq2}) входят неизвестные числовые коэффициенты: ${C_{kn}},{D_{km}}$ --- произвольные постоянные интегрирования,
определяемые из граничных условий, и~значения ${w_{my}}\left( {{y_i}} \right)\!,~{w_{nx}}\left( {{x_j}} \right)$.
Для нахождения указанных числовых коэффициентов вычислим коэф\-фи\-циен\-ты Фурье решения (\ref{eq2}):

\[{w_{nx}}\left( x \right) = \sum\limits_{m = 1}^\infty  {\frac{{{q_{mn}}\sin \left( {{\alpha _m}x} \right)}}{{D{{\left( {\alpha _m^2 +
 \beta _n^2} \right)}^2}}}} - \sum\limits_{k = 1}^4 {{C_{kn}}{Z_{{\beta _n}k}}\left( x \right) - } \]
\[ - \sum\limits_{m = 1}^\infty  {\sum\limits_{k = 1}^4 {{D_{km}}\bar Z_{{\alpha _m}k}^{{\beta _n}}\sin \left( {{\alpha _m}x} \right) ~ - } } \, \]
\[ - \sum\limits_{m = 1}^\infty  {\sum\limits_{i = 1}^{{K_1}} {\left[ {{\lambda _{1i}}\alpha _m^4 +
 \frac{{{P_{1i}}}}{D}\alpha _m^2} \right]{w_{my}}\left( {{y_i}} \right)\bar \Psi _{_{i{\alpha _m}}}^{*{\beta _n}}\sin \left( {{\alpha _m}x} \right)} } - ~\, \]
\begin{equation}
\label{eq3}
 - \sum\limits_{j = 1}^{{K_2}} {\left[ {{\lambda _{2j}}\beta _n^4 +
 \frac{{{P_{2j}}}}{D}\beta _n^2} \right]{w_{nx}}\left( {{x_j}} \right)\Psi _{j{\beta _n}}^*\left( x \right)} ,
\end{equation}
\[{w_{my}}\left( y \right) = \sum\limits_{n = 1}^\infty  {\frac{{{q_{mn}}\sin \left( {{\beta _n}y} \right)}}{{D{{\left( {\alpha _m^2 +
 \beta _n^2} \right)}^2}}}}  - \sum\limits_{n = 1}^\infty
  {\sum\limits_{k = 1}^4 {{C_{kn}}\bar Z_{{\beta _n}k}^{{\alpha _m}} %\left( x \right)%
  \sin \left( {{\beta _n}y} \right) - } } \]
\[ - \sum\limits_{k = 1}^4 {{D_{km}}{Z_{{\alpha _m}k}}(y) - } \sum\limits_{i = 1}^{{K_1}} {\left[ {{\lambda _{1i}}\alpha _m^4 +
\frac{{{P_{1i}}}}{D}\alpha _m^2} \right]{w_{my}}\left( {{y_i}} \right)\Psi _{_{i{\alpha _m}}}^*\left( y \right)} \, - \]
\begin{equation}
\label{eq4}
 - \sum\limits_{j = 1}^{{K_2}} {\sum\limits_{n = 1}^\infty  {\left[ {{\lambda _{2j}}\beta _n^4 +
 \frac{{{P_{2j}}}}{D}\beta _n^2} \right]{w_{nx}}\left( {{x_j}} \right)\bar \Psi _{j{\beta _n}}^{*{\alpha _m}}\sin \left( {{\beta _n}y} \right)} } .
\end{equation}
Полагая в~(\ref{eq3}), что $x = {x_p}, ~p = 1,2,\,\ldots,\,{K_2}$, а~в~ (\ref{eq4}) --- что $y = {y_q}, q = 1,2,\,\ldots,\,{K_1}$,
получим две группы уравнений
\[{w_{nx}}\left( {{x_p}} \right) = \sum\limits_{m = 1}^\infty
 {\frac{{{q_{mn}}\sin \left( {{\alpha _m}{x_p}} \right)}}{{D{{\left( {\alpha _m^2 + \beta _n^2} \right)}^2}}}}  -
 \sum\limits_{k = 1}^4 {{C_{kn}}{Z_{{\beta _n}k}}\left( {{x_p}} \right) - } \]
\[ - \sum\limits_{m = 1}^\infty  {\sum\limits_{k = 1}^4 {{D_{km}}\bar Z_{{\alpha _m}k}^{{\beta _n}}\sin \left( {{\alpha _m}{x_p}} \right) - } } \]
\[ - \sum\limits_{m = 1}^\infty  \sum\limits_{i = 1}^{{K_1}}
\left[{\lambda _{1i}}\alpha _m^4}+{\frac{{P_{1i}}}{D}\alpha _m^2}\right] {w_{my}}\left( {{y_i}} \right)\bar \Psi _{_{i{\alpha _m}}}^{*{\beta _n}}\sin \left( {{\alpha _m}{x_p}} \right)}  \, - \]
\begin{equation}
\label{eq5}
 - \sum\limits_{j = 1}^{{K_2}} {\left[ {{\lambda _{2j}}\beta _n^4 +
\frac{{P_{2j}}}{D}\beta _n^2} \right]{w_{nx}}\left( {{x_j}} \right)\Psi _{j{\beta _n}}^*\left( {{x_p}} \right)} ,
\end{equation}

\noindent
\[{w_{my}}\left( {{y_q}} \right) = \sum\limits_{n = 1}^\infty  {\frac{{{q_{mn}}\sin \left( {{\beta _n}{y_q}} \right)}}{{D{{\left( {\alpha _m^2 +
\beta _n^2} \right)}^2}}}}  - \sum\limits_{n = 1}^\infty  {\sum\limits_{k = 1}^4 {{C_{kn}}\bar Z_{{\beta _n}k}^{{\alpha _m}} %\left( x \right)%
\sin \left( {{\beta _n}{y_q}} \right) - } } \]
\[ - \sum\limits_{k = 1}^4 {{D_{km}}{Z_{{\alpha _m}k}}\left( {{y_q}} \right) - }
 \sum\limits_{i = 1}^{{K_1}} \left[{{\lambda _{1i}}\alpha _m^4 +
\frac{{{P_{1i}}}}{D}\alpha _m^2} \right]{w_{my}}\left( {{y_i}} \right)\Psi _{_{i{\alpha _m}}}^*\left( {{y_q}} \right)} \, - \]
\begin{equation}
\label{eq6}
 - \sum\limits_{j = 1}^{{K_2}} {\sum\limits_{n = 1}^\infty  {\left[ {{\lambda _{2j}}\beta _n^4 -
 \frac{{P_{2j}}}{D}\beta _n^2} \right]{w_{nx}}\left( {{x_j}} \right)\bar \Psi _{j{\beta _n}}^{*{\alpha _m}}\sin \left( {{\beta _n}{y_q}} \right)}} ,
\end{equation}
где введены обозначения коэффициентов Фурье
\[\bar Z_{{\alpha _m}k}^{{\beta _n}} = \frac{2}{b}\int\limits_0^b {{Z_{{\alpha _m}k}}\left( y \right)\sin \left( {{\beta _n}y} \right)dy,} {\rm{ }}~~
\bar Z_{{\beta _n}k}^{{\alpha _m}} = \frac{2}{a}\int\limits_0^a {{Z_{{\beta _n}k}}\left( x \right)\sin \left( {{\alpha _m}x} \right)dx} ,\]
\[\bar \Psi _{i{\alpha _m}}^{*{\beta _n}} = \frac{2}{b}\int\limits_0^b {\Psi _{i{\alpha _m}}^*\left( y \right)\sin \left( {{\beta _n}y} \right)dy} ,{\rm{ }}~~
\bar \Psi _{j{\beta _n}}^{*{\alpha _m}} = \frac{2}{a}\int\limits_0^a {\Psi _{j{\beta _n}}^*\left( x \right)\sin \left( {{\alpha _m}x} \right)dx} .\]

\noindent
К (\ref{eq5}) и~(\ref{eq6}) необходимо добавить уравнения, полученные из граничных условий.
Явно выписывать последние не будем.
Таким образом, решение задачи определяется функцией (\ref{eq2}), в~которой числовые коэффициенты находятся
как решения линейной алгебраической системы уравнений, получаемой по стандартному алгоритму.

\noindent

\textbf{3. Пример.} Рассмотрим следующую задачу. Пусть прямоугольная пластина
$0 \le x \le 6 м$, $0 \le y \le 4 м$ нагружена поперечной нагрузкой
$q\left( {x,y} \right) = {p_0} = 40$~кН/м$^{2}$ и~подкреплена ребрами,
расположенными параллельно осям $x$ и~$y$ по линиям ${x_1} = 1.5$~м, ${x_2} = 3$ м,
${x_3} = 4.5 $~м; ${y_1} = {4 \mathord{\left/
 {\vphantom {4 3}} \right.
 \kern-\nulldelimiterspace} 3}$ м,
${y_2} = {8 \mathord{\left/
 {\vphantom {8 3}} \right.
 \kern-\nulldelimiterspace} 3}$ м.
Обшивка пластины имеет постоянную толщину $h = 0.012 м$. Ребра жесткости:
горизонтальные ригели — тавры: высота стенки~—~0.8~м, толщина стенки~— 0.02 м,
ширина полки~— 0.35 м, толщина полки~— 0.04~м; вертикальные стойки — тавры:
высота стенки~— 0.6 м, толщина стенки~— 0.02~м, ширина полки~— 0.35 м, толщина полки~— 0.03 м.
Для определенности будем считать, что пластина шарнирно оперта по контуру. На пластину, кроме поперечной нагрузки
$q\left(x,y\right)$, действуют растягивающие усилия $P = 30$ кН по кромкам в~плос\-кос\-ти пластины,
которые передаются через ребра параллельно оси $0y$ (рис. 1).

\textbf{4. Заключение.} Результаты расчетов ребристых пластин показывают достаточно хорошую сходимость рядов в~полученных решениях.
Однако следует отметить, что сходимость рядов ухудшается на линиях расположения ребер жесткости.
Таким образом, для расчета НДС непосредственно в~ребре следует удерживать значительное количество членов в~рядах.
Ухудшение сходимости полученных рядов непосредственно связано с~жесткостью ребер: чем больше жесткость ребра,
тем большее число членов следует удерживать в~рядах для получения приемлемого результата.
При малых жесткостях ребер ряды хорошо сходятся во всей области, в~том числе и~на линиях расположения ребер.
С увеличением числа удерживаемых членов в~рядах проявляется еще одна особенность рассматриваемых задач
 --- вычислительная неустойчивость, которая также непосредственно связана с~жесткостью ребер.
Это приводит к~тому, что расчет НДС в~ребре оказывается совершенно невозможен на основе общих формул.
Предложена формула в~виде одинарного ряда для вычисления НДС в~ребрах жесткости,
обладающая хорошей сходимостью на сингулярных линиях, как и~общая формула вне этих линий.
