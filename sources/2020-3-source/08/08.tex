{\leftskip=7mm\noindentРассматривается управляемая система
линейных дифференциально-разностных уравнений с~линейно
возрастающим запаздыванием. Известны достаточные условия
асимптотической устойчивости таких систем, однако общие условия
стабилизируемости управляемых систем и~конструктивные алгоритмы
построения стабилизирующих управлений до сих пор не были получены.
Для линейной дифферен\-циально-разностной системы запаздывающего
типа с~линейно возрастающим запаздыванием применяется каноническое
преобразование Зубова и~выводятся условия стабилизируемости этих
систем статическим управлением. Сформулирован алгоритм проверки
условий существования стабилизирующего управления и~его
построения. Доказаны новые теоремы об анализе устойчивости систем
линейных диф\-фе\-рен\-циально-разностных уравнений с~линейно
возрастающим запаздыванием. Полученные результаты могут быть
перенесены на случай систем с~несколькими пропорциональными
сосредоточенными запаз\-ды\-ва\-ниями.\\[1mm]
\textit{Ключевые слова}: система дифференциально-разностных
уравнений, линейное запаздывание, асимптотическая устойчивость,
стабилизирующее управление, асимптотический наблюдатель.

}

}

{\bfseries {1.} Введение.} Системы дифференциально-разностных
уравнений с~запаздыванием в~настоящее время находятся в~центре
внимания многих исследователей. Необходимость глубокого анализа
таких систем связана в~том числе с~целым рядом математических
моделей, учитывающих запаздывающий аргумент. В~качестве примера
этих моделей следует отметить модель радиоактивного распада [1],
работу информационного сервера [2], распространение эпидемии [3]
и~т.~д.

Естественным обобщением систем с~постоянным запаздыванием являются
системы с~запаздыванием, линейно зависящим от времени. К~важным
проблемам при их изучении относится проблема устойчивости. Наличие
в~системе неограниченного запаздывания может привести к~потере
устойчивости, что обусловливает пристальное внимание  к~этой
проблеме. В~работах [2--5]  получены достаточные условия
асимптотической устойчивости систем с~одним или несколькими
запаздываниями, линейно зависящими от времени.

Линейные управляемые системы с~запаздыванием рассмотрены в~[6--8].
В~работе [6] представлены некоторые методы стабилизации линейных
управляемых систем с~постоянными матрицами и~линейными
запаздываниями, в~[7]~---  алгоритм построения стабилизирующего
управления для линейных систем с~запаздыванием в~случае неполной
информации. Проблема стабилизации линейных систем
с~пропорциональными запаздываниями рассмотрена в~[8]. Получены
достаточные условия существования стабилизирующего управления
в~случаях полной и~неполной информации о векторе состояния
системы.

В~данной работе будем использовать каноническое преобразование
Зубова [9], адаптированное для изучаемых линейных управляемых
систем с~линейно возрастающим запаздыванием. Основываясь на
каноническом виде приведенной системы, выведены условия
стабилизируемости, уточняющие результаты  [8]. Выявлены условия
стабилизируемости системы статическим управлением, а~также
с~ис\-поль\-зова\-нием асимптотического наблюдателя. Все
доказательства являются конструктивными и~легко
ал\-го\-рит\-ми\-зи\-руют\-ся, что проиллюстрировано в~примере.

%-------------------------------------- 2. постановка задачи -------------------------------------------
%\paragraph {\indent \bfseries {\itshape 2.} Постановка задачи.}
{\bfseries\indent \bfseries {2.} Постановка задачи.} В~системе
уравнений
\begin{equation}\label{eq1} %                                           (1)
    \dot{x}(t) = A_0 x(t) + A_1 x(\alpha t) + B u,
\end{equation}   % eq 1
\begin{equation}\label{eq2} %                                           (2)
    y(t) = K x(t)
\end{equation}     % eq 2
$x$~---  $n$-мерный вектор состояния, $u$~---  $r$-мерный вектор
управления, $y$~---  $q$-мерный вектор наблюдений, $A_0, A_1,B,K$
--- постоянные матрицы размерностей $ (n \times n), (n \times r) $
и~$(q \times n)$ соответственно, $0<\alpha <1$. Необходимо
построить управление $u = u \left( x(t),x(\alpha t) \right)$,
стабилизирующее систему (\ref{eq1}). Это означает, что нулевое
решение замк\-ну\-той этим управлением системы (\ref{eq1})
асимптотически устойчиво по Ляпунову. Будем придерживаться
следующих определений.

Введем в~рассмотрение начальную функцию  $ \phi (t), t \in [\alpha
t_0,t_0] $, такую, что $ \phi (t_0)=x_0, t_0 > 0.$ Очевидно,
нулевое решение системы (\ref{eq1}) соответствует нулевой
начальной функции.

%\newtheorem{definition}{\indent Определение}

\begin{opredelenie}\label{D1} % Определение 1
Нулевое решение системы {\normalfont (\ref{eq1})} устойчиво по
Ляпунову, если для любого числа $ \epsilon > 0 $ и~любого
начального момента $ t_0 > 0 $ можно указать такое число $
\delta(\epsilon,t_0) > 0, $ что для всякой начальной функции  $
\phi(t), $ для которой $ \| \phi(t) \| < \delta$ при  $ t \in
[\alpha t_0,t_0], $ решение системы {\normalfont (\ref{eq1})} $ x
= x(t,t_0, \phi) $  удовлетворяет условию $ \| x(t,t_0, \phi) \| <
\epsilon $  для любого $ t \ge t_0.$

\end{opredelenie}

\begin{opredelenie}\label{D2} % Определение 2
Нулевое решение системы {\normalfont (\ref{eq1})} асимптотически
устойчиво по Ляпунову, если оно устойчиво по Ляпунову в~смысле
определения 1, и~для любого $ t_0 > 0 $ можно указать такое число
$\delta_1 > 0,$ что для всякой начальной функции  $ \phi(t), $ для
которой $ \| \phi(t) \| < \delta_1$  при $ t \in   [\alpha
t_0,t_0], $ решение системы  {\normalfont (\ref{eq1})} $ x =
x(t,t_0, \phi) $ удовлетворяет условию $ \| x(t,t_0, \phi) \| \to
0 $  при $ t \to +\infty.$


\end{opredelenie}
В данной статье предлагается решение задачи синтеза
стабилизирующего управления  в~случае как полной, так и~неполной
информации о~векторе состояния системы. В~случае полной информации
целью работы является получение условий на матрицы  $A_0, A_1, B$,
выполнение которых обеспечивает решение задачи стабилизации.
В~случае неполной информации строится система асимптотической
оценки на основе наблюдения (\ref{eq2}), которая используется для
построения стабилизирующего управления.

%------------------------------- 3. Скалярное управление в~случае полной информации.--------------
%\paragraph {\indent \bfseries {\itshape 3.} Скалярное управление в~случае полной информации.}
{\indent \bfseries {3.} Скалярное управление в~случае полной
информации.} Рассмотрим систему (\ref{eq1}) при скалярном
управлении
$$  \dot{x}(t) = A_0 x(t) + A_1 x(\alpha t) + b u $$
и управление
\begin{equation}\label{eq3} %                                           (3)
    u = c_0x(t) + c_1x(\alpha t).
\end{equation}  % eq 3
Согласно [10], известно, что замкнутая система (\ref{eq1}),
(\ref{eq3}) асимптотически устойчива, если все корни уравнения
%\begin{equation}\label{eq4}%                                                                    (4)
$$
    \det(\lambda E -A_0-bc_0) = 0
 $$
 %\end{equation}          % eq 4
лежат в~левой полуплоскости комплексной плоскости, а~корни
уравнения

\begin{equation}\label{eq5}
    \det [(A_0+bc_0) + \rho (A_1 + bc_1)] = 0
\end{equation}  % eq 5%                                         (5)
по модулю больше единицы.


% \newtheorem{lemma}[]{\indent Лемма}

 % -------------------------------------------- Лемма 1 ------------------------------------------
\begin{lemmar}\label{L1}
Предположим, что матрица $A_0$  не вырождена. Если матрицы
$$
P_0 = (b; A_0 b; \ldots ; A_0^{n-1}b) \mbox{  ~и ~}  P_1 =
[A_0^{-1}b; A_0^{-1} A_1 A_0^{-1} b; \ldots ;
(A_0^{-1}A_1)^{n-1}A_0^{-1}b]
$$
удовлетворяют условиям {$\textnormal {rang} (P_0) = \textnormal
{rang} (P_1) = n$}, то существуют векторы $c_0$ и~$c_1$ такие, что
управление {\normalfont (\ref {eq3})} обеспечивает
стабилизируемость системы {\normalfont(\ref{eq1})}.
\end{lemmar}

{\bf Пример 1.} Рассмотрим линейную управляемую систему с~линейным
запаздыванием
$$
\dot{x}(t) = \left (\begin{array}{rrr} 1 & -2 & 0 \\ \beta & -2 &
0 \\ 1 & 1 & 2 \end{array} \right ) x(t) + \left
(\begin{array}{rrr} 0 & 0 & 1 \\ 0 & 1 & 1 \\ 1 & 1 & 0
\end{array} \right ) x(\alpha t) + \left (\begin{array}{r} 1  \\
0 \\ 0 \end{array} \right ) u.
$$
Здесь $ A_0 = \left (\begin{array}{rrr} 1 & -2 & 0 \\ \beta & -2 &
0 \\ 1 & 1 & 2 \end{array} \right ),$ $ A_1 = \left
(\begin{array}{rrr} 0 & 0 & 1 \\ 0 & 1 & 1 \\ 1 & 1 & 0
\end{array} \right ),$ $b=  \left (\begin{array}{r} 1  \\  0 \\ 0
\end{array} \right ),$ где $\beta$~---  вещественный параметр.

Матрица $A_0$ невырожденная почти при всех значениях параметра
$\beta$, за исключением $\beta~=~1.$ Положим $\beta=2$,
и~исследуем стабилизируемость системы в~соответствии с~леммой~1:
$$
    A_0 = \left (\begin{array}{rrr} 1 & -2 & 0 \\ 2 & -2 & 0 \\ 1 & 1 & 2 \end{array} \right ), ~A_0^{-1}A_1 = \left (\begin{array}{rrr} 0 & 1 & 0 \\ 0 & 0.5 & -0.5 \\ 0.5 & -0.25 & 0.25 \end{array} \right ), ~P_1 = \left (\begin{array}{rrr} -1 & -1 & -1 \\ -1 & -1 & -0.5 \\ 1 & 0 & -0.25 \end{array} \right ).
$$

с~леммой 2 заданная система стабилизируется управлением

полагаем матрицу $ \tilde A_0 = A_0 + BC_0 $ гурвицевой.

$ L = \{D\} $~---  линейное пространство, натянутое на столбцы

\begin{lemmar}\label{L3}
Пусть  $\hat C_0$~---  произвольная матрица такая, что $ \hat A_0
= \tilde A_0 + B \hat C_0$~--- невырожденная матрица. Рассмотрим
пространства
 $$
 \tilde L = \{\tilde{A}_0^{-1}B;  \tilde{A}_0^{-1} A_1 \tilde{A}_0^{-1}B; \ldots; (\tilde{A}_0^{-1} A_1)^{n-1} \tilde{A}_0^{-1} B \},
 $$
$$
\hat L = \{\hat{A}_0^{-1}B;  \hat{A}_0^{-1} A_1 \hat{A}_0^{-1}B;
\ldots; (\hat{A}_0^{-1} A_1)^{n-1} \hat{A}_0^{-1} B \}.
$$

       Тогда справедливо равенство  $   \hat L = \tilde L  $.
\end{lemmar}   % end Лемма 3----------------

{\bf 5. Случай неполной информации. Асимптотический наблюдатель.}
Построение асимптотической оценки будем вести стандартным образом,
опираясь на особенности системы и~расширяя возможности подхода
работы [8]. Рассмoтрим систему уравнений (\ref{eq1}), наблюдение
(\ref{eq2}) и~будем строить асимптотическую оценку вектора
состояния  в~виде системы
\begin{equation}\label{eq13} %                                          (13)
\dot{\hat x}(t) = A_0 \hat x(t) + A_1 \hat x(\alpha t) + B u +
L_0(K \hat x(t) - y(t)) +
    L_1(K\hat x(\alpha t) - y(\alpha t)).
\end{equation}
Чтобы найти матрицы $L_0$ и~$L_1$, рассмотрим сопряженную систему
\begin{equation}\label{eq14} %                                          (14)
    \dot{z}(t) = A_0^T z(t) + A_1^T z(\alpha t) + K^Tv.
\end{equation}
Если система (\ref{eq14}) удовлетворяет условиям теоремы 2, то
существует стабилизирующее эту систему управление $ v = \hat C_0
z(t) + \hat C_1 z(\alpha t).$ Если теперь положить, что $L_0 =
C_0^T$ и~$L_1 = C_1^T,$ то система
$$
    \dot{\tilde x}(t) = A_0 \tilde x(t) + A_1 \tilde x(\alpha t) + L_0K \tilde x(t) +
    L_1K\tilde x(\alpha t)
$$
будет асимптотически устойчивой.

       Заметим, что $\tilde x(t) = \hat x(t) - x(t),$ поэтому $\hat x(t) \rightarrow x(t)$ при $t \rightarrow +\infty$ и~$\tilde x(t) \rightarrow 0$.

       Предположим, что управление $ u = C_0  x(t) + C_1  x(\alpha t)$ стабилизирует исходную систему (\ref{eq1}), но будем использовать управление
\begin{equation}\label{eq15} %                                          (15)
     u = C_0 \hat x(t) + C_1 \hat x(\alpha t).
\end{equation}
Представим систему (\ref{eq1}) в~виде
$$
    \dot{x}(t) = (A_0 +BC_0) x(t) + (A_1 + BC_1)  x(\alpha t) + BC_0 \tilde x(t) + BC_1 \tilde x(\alpha t).
$$
Поскольку в~этой линейной системе однородная часть асимптотически
устойчива, а~неоднородность стремится к~нулю, то, очевидно,
система (\ref{eq1}), (\ref{eq13}), (\ref{eq15}) асимптотически
устойчива по Ляпунову. Таким образом справедлива
% -------------------------------------------- Теорема 3 -------------------------------------------------------------

\begin{teoremar} \label{T3}
Если системы {\normalfont (\ref{eq1})} и~ {\normalfont
(\ref{eq14})} удовлетворяют условиям теоремы  2, то существует
стабилизирующее управление {\normalfont (\ref{eq13})},
{\normalfont (\ref{eq15})}.
\end{teoremar}     % end Teorema 3
% -------------------------------------------- Теорема 3~---  Доказательство ---------------------------------------
%\begin{proof}[\indent \normalfont

Д~о~к~а~з~а~т~е~л~ь~с~т~в~о%]
. Доказательство приведено выше и~является конструктивным
алгоритмом построения стабилизирующего управления.\hfill\square
%\end{proof}

Этот {\bfseriesалгоритм} можно расписать следующим образом.

\indent {\itshape Шаг 1.} Проверяем необходимые условия стабилизации, сформулированные в~тео\-реме~2.\\
\indent {\itshape Шаг 2.} Проверяем выполнение условий теоремы 2 для системы (10).\\
\indent {\itshape Шаг 3.} Если проверка прошла успешно, то
вычисляем матрицы  и~выписываем стабилизирующее управление в~виде
системы (\ref{eq13}), (\ref{eq15}).
