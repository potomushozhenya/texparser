{\leftskip=7mm\noindentДанная работа посвящена моделированию
полевой эмиссионной диодной системы. Поверхность эмиттера ---
гиперболоид вращения.  Поверхность анода --- часть гиперболоида
вращения, в частном случае --- круговая диафрагма. Поставлена
граничная задача для уравнения Лапласа с граничными условиями
первого рода, не обладающими осевой симметрией. 3D-решение задачи
найдено методом разделения переменных в вырожденных
эллипсоидальных координатах для вытянутого эллипсоида вращения.
Распределение электростатического потенциала представлено в виде
разложений по функциям Лежандра. Вычисление коэффициентов
разложений сведено к решению системы линейных алгебраических
уравнений с постоянными коэффициентами. Все геометрические размеры
системы являются параметрами задачи.\\[1mm]
\textit{Ключевые слова}: микро- и наноэлектроника, полевой
эмиттер, полевая эмиссия, математическое моделирование,
электростатический потенциал, граничная задача, функции Лежандра.

}

}

{\bf 1. Введение.} Полевые эмиттеры по сравнению с термо- и
фотокатодами обладают многочисленными преимуществами, например,
такими, как высокая яркость, монокинетичность пучка, работа при
комнатных температурах, низкое энергопотребление, малый вес и
габариты и т.~д.  [1--3]. Эти особенности источников, основанных
на принципе полевой эмиссии, позволяют использовать их в различных
вакуумных приборах микро- и наноэлектроники [4--7].

В данной статье моделируется диодная система с полевым эмиттером в
виде гиперболоида вращения. Поверхность анода представляет собой
часть гиперболоида вращения или круговую диафрагму. Для вычисления
электростатического по\-тен\-циа\-ла $U(\alpha,\beta,\phi)$ во
всей области системы используется метод разделения переменных в
координатах вытянутого эллипсоида вращения ($\alpha,\beta,\phi$).
Таким образом, поверхности электродов являются осесимметричными
(рисунок). Граничные условия первого рода на электродах не
обладают свойством осевой симметрии и зависят от угловой
координаты~$\phi$. %\vskip 0.7cm

%\noindent
{\bf 2. Математическая модель.  }
%
Параметры задачи:

$\beta=\beta_1,\;\; 0\le \alpha < \infty,\;\; -\pi\le \phi < \pi$
---  поверхность катода;

$\alpha=\alpha_0,\beta=\beta_0$ --- координаты круговой апертуры
анода;

$\beta=\beta_0,\;\; \alpha_0\le \alpha < \infty,\;\; -\pi\le \phi
< \pi$ --- поверхность анода;

$U(\alpha, \beta_1,\phi)=f_1(\alpha, \phi), \;\; 0\le \alpha <
\infty,\;\; -\pi\le \phi < \pi $ --- граничное условие на катоде;

$U(\alpha, \beta_0,\phi)=f_0(\alpha, \phi), \;\; \alpha_0\le
\alpha < \infty,\;\; -\pi\le \phi < \pi $ --- граничное условие на
аноде.

При $\beta_0=\pi/2$ поверхность анода --- плоская круговая
диафрагма,  при $\beta_0\ne \pi/2$~--- часть гиперболоида
вращения.


%{\footnotetext{Вестник СПбГУ. Прикладная математика. Информатика... 2020. Т. 16. Вып. 3}}

%\vskip .3cm


3D-распределение электростатического потенциала
$U(\alpha,\beta,\phi)$ удовлетворяет урав\-не\-нию Лапласа
\begin{eqnarray}
\begin{array}{c}
\displaystyle \frac{1}{c^2(\sin^2\beta + \sinh^2\alpha)} \Bigg[
\displaystyle \dfrac{\partial^2 U}{\partial\alpha^2} + \coth\alpha
\displaystyle \dfrac{\partial U}{\partial\alpha} + \displaystyle
\dfrac{\partial^2 U}{\partial\beta^2}+ \cot\beta\displaystyle
\dfrac{\partial U}{\partial\beta}
\, +  \\[7mm]
+\left(\displaystyle \frac{1}{\sinh^2\alpha}+
\dfrac{1}{\sin^2\beta }\right)\dfrac{\partial^2 U}{\partial\phi^2}
\Bigg]   = 0,
 \\[7mm]
%\hspace*{1.7cm}
 0\le \alpha < \infty, \  0\le \beta \le \beta_1, \
-\pi\le \phi < \pi,
\end{array}
\end{eqnarray}%1
и граничным условиям
\begin{eqnarray}
\begin{array}{ll}
U(\alpha, \beta_1,\phi) = f_1(\alpha,\phi),&   0\le \alpha <
\infty,\ -\pi\le \phi < \pi,
\mbox{\rule[-3mm]{0mm}{5mm}}    \\
U(\alpha, \beta_0,\phi) = f_0(\alpha,\phi),&   \alpha_0\le \alpha
< \infty,\ -\pi\le \phi < \pi,
\end{array}
\end{eqnarray}%2
где $c$ --- фокальный радиус.


%\vskip 0.3cm

{\bf 3. Вычисление электростатического потенциала. } Для решения
граничной задачи (1), (2) рассмотрим следующие области:

\vskip .1cm


{\bf 1} --- $0\le \alpha < \infty,\ \ \beta_0\le \beta\le
\beta_1,\ \ -\pi\le \phi < \pi$;

{\bf 2} --- $0\le \alpha < \infty,\ \ 0\le \beta\le \beta_0,\ \ \
-\pi\le \phi < \pi $;

{\bf 3} --- $ 0\le \alpha < \alpha_0,\ \ 0\le \beta\le \beta_1,\ \
\ -\pi\le \phi < \pi $.

%\vskip .1cm

\noindent Обозначим распределение потенциала в {\it j}\,-й области
$U(\alpha,\beta,\phi)=U_j(\alpha,\beta,\phi)$, $j=1,2,3$. Тогда
для каждой {\it j}\,-й области функцию $U_j(\alpha,\beta,\phi)$
соответственно можно представить в виде [8--12]

\begin{eqnarray}
\begin{array}{c}
\displaystyle
U_1(\alpha,\beta,\phi)=\sum\limits_{m=-\infty}^{\infty}
e^{im\phi}\, \int\limits_{0}^{\infty} \Bigg[ A_m(\tau)
\dfrac{\displaystyle
W^m_{-\frac{1}{2}+i\,\tau}(\cos\beta,\cos\beta_0)}{\displaystyle
W^m_{-\frac{1}{2}+i\,\tau}(\cos\beta_1,\cos\beta_0)} \,+
 \\[9mm]
 \displaystyle %\hspace*{3cm}
+\, B_m(\tau)
 \dfrac{\displaystyle W^m_{-\frac{1}{2}+i\,\tau}(\cos\beta_1,\cos\beta)}{\displaystyle  W^m_{-\frac{1}{2}+i\,\tau}(\cos\beta_1,\cos\beta_0)}\Bigg] P^m_{-\frac{1}{2}+i\,\tau}(\cosh\alpha)\, d\tau,
%  \\[9mm]
\end{array}
\end{eqnarray}%3

\begin{eqnarray}
\begin{array}{l}
  \displaystyle
U_2(\alpha,\beta,\phi)=\sum\limits_{m=-\infty}^{\infty}
e^{im\phi}\, \int\limits_{0}^{\infty}B_m(\tau)
\dfrac{\displaystyle
P^m_{-\frac{1}{2}+i\,\tau}(\cos\beta)}{\displaystyle
P^m_{-\frac{1}{2}+i\,\tau}(\cos\beta_0)}
P^m_{-\frac{1}{2}+i\,\tau}(\cosh\alpha)\, d\tau,
% \\[9mm]
\end{array}
\end{eqnarray}%4

\begin{eqnarray}
\begin{array}{c}
U_3(\alpha,\beta,\phi)=\sum\limits_{m=-\infty}^{\infty}
e^{im\phi}\, \Bigg[\sum\limits_{n=1}^{\infty}C_{m,n}
\dfrac{\displaystyle
P^m_{-\frac{1}{2}+i\,\nu_{m,n}}(\cos\beta)}{\displaystyle
P^m_{-\frac{1}{2}+i\,\nu_{m,n}}(\cos\beta_1)}
P^m_{-\frac{1}{2}+i\,\nu_{m,n}}(\cosh\alpha) \,+
 \\[7mm]  \displaystyle %\hspace*{5cm}
+\, \sum\limits_{k=1}^{\infty}D_{m,k}
 \dfrac{\displaystyle P^m_{\mu_{m,k}}(\cosh\alpha)}{\displaystyle  P^m_{\mu_{m,k}}(\cosh\alpha_0)} P^m_{\mu_{m,k}}(\cos\beta)\Bigg],
\end{array}
\end{eqnarray}%5
%
здесь $P^m_{\sigma}(z)$ --- присоединенные функции Лежандра
первого рода, $Q^m_{\rho}(z)$
 --- при\-сое\-ди\-нен\-ные функции Лежандра второго рода,
$W^m_{-\frac{1}{2}+i\,\tau}(x, y)$ --- линейная комбинация функций
Лежандра
\begin{eqnarray}
W^m_{-\frac{1}{2}+i\,\tau}(x,y)= P^m_{-\frac{1}{2}+i\,\tau}( x)\,
Q^m_{-\frac{1}{2}+i\,\tau}( y)- P^m_{-\frac{1}{2}+i\,\tau}( y)\,
Q^m_{-\frac{1}{2}+i\,\tau}( x),
\end{eqnarray}%6
$\nu_{m,n}$, $\mu_{m,k}$ --- корни уравнений

\begin{eqnarray}
P^m_{-\frac{1}{2}+i\,\nu_{m,n}}(\cosh\alpha_0)=0, \ \ \ \
P^m_{\mu_{m,k}}(\cos\beta_1)=0.
\end{eqnarray}%7
%{\footnotetext{Вестник СПбГУ. Прикладная математика. Информатика... 2020. Т. 16. Вып. 3}}

\noindent Для отрицательных значений $m$ в (3)--(5) при
$x=\cos\beta$, $z=\cosh\alpha$ справедливы сле\-дующие формулы
[10, 13, 14]:
$$
P^{-m}_{\sigma}(x)=(-1)^m\dfrac{\Gamma(\sigma-m+1)}{\Gamma(\sigma+m+1)}
P^m_{\sigma}(x),\ \ \
Q^{-m}_{\sigma}(x)=(-1)^m\dfrac{\Gamma(\sigma-m+1)}{\Gamma(\sigma+m+1)}
Q^m_{\sigma}(x),
$$
$$
P^{-m}_{\sigma}(z)=\dfrac{\Gamma(\sigma-m+1)}{\Gamma(\sigma+m+1)}
P^m_{\sigma}(z),\ \ \
Q^{-m}_{\sigma}(z)=\dfrac{\Gamma(\sigma-m+1)}{\Gamma(\sigma+m+1)}
Q^m_{\sigma}(z),
$$
где $\Gamma(y)$ --- гамма-функция.

Следует отметить, что формулы (3) и (4) обеспечивают непрерывность
распределения потенциала $U(\alpha,\beta,\phi)$ во всем
пространстве исследуемой системы в силу того, что на границе
раздела областей {\bf 1} и {\bf 2} при $\beta=\beta_0$ и $0\le
\alpha<\infty$ выполняется условие
%
\begin{eqnarray}
U_1(\alpha,\beta_0,\phi)=U_2(\alpha,\beta_0,\phi)=
\sum\limits_{m=-\infty}^{\infty} e^{im\phi}\,
\int\limits_{0}^{\infty}B_m(\tau)
 P^m_{-\frac{1}{2}+i\,\tau}(\cosh\alpha)\, d\tau.
\end{eqnarray}%8

\noindent Коэффициенты $A_m(\tau)$ в разложении (3) рассчитываются
в явном виде из граничных условий (2) на катоде при
$\beta=\beta_1$, $0\le \alpha<\infty$:
$$
U_1(\alpha, \beta_1,\phi) = f_1(\alpha,\phi)=
\sum\limits_{m=-\infty}^{\infty} e^{im\phi}\,
\int\limits_{0}^{\infty}
A_m(\tau)\,P^m_{-\frac{1}{2}+i\,\tau}(\cosh\alpha)\, d\tau,
$$
с помощью интегрального преобразования Мелера---Фока [8, 9]
\begin{eqnarray}
 A_m(\tau)=\dfrac{\displaystyle (-1)^m}{\displaystyle 2\pi} \tau  \tanh(\pi \tau)
 \int\limits_{-\pi}^{\pi} e^{-im\phi}\,
 \int\limits_{0}^{\infty}f_1(\alpha,\phi) \,P^{-m}_{-\frac{1}{2}+i\,\tau}(\cosh\alpha)\, \sinh\alpha\, d\alpha \, d\phi.
\end{eqnarray}%9

\noindent Коэффициенты $C_{m,n}$ в разложении (5) также
вычисляются в явном виде из граничных условий (2) при
$\beta=\beta_1$, $0\le \alpha\le \alpha_0$:
$$
U_3(\alpha,\beta_1,\phi) =
f_1(\alpha,\phi)=\sum\limits_{m=-\infty}^{\infty} e^{im\phi}\,
\sum\limits_{n=1}^{\infty}C_{m,n}
P^m_{-\frac{1}{2}+i\,\nu_{m,n}}(\cosh\alpha)
$$
и из условий ортогональности системы собственных функций
$P^m_{-\frac{1}{2}+i\,\nu_{m,n}}(\cosh\alpha)$
\begin{eqnarray}
C_{m,n}=\frac{1}{2\pi N_{m,n}^1} \int\limits_{-\pi}^{\pi}
e^{-im\phi}\, \int\limits_{0}^{\alpha_0}f_1(\alpha,\phi)
\,P^{-m}_{-\frac{1}{2}+i\,\nu_{m,n}}(\cosh\alpha)\, \sinh\alpha\,
d\alpha \, d\phi.
\end{eqnarray}%10

\noindent Нормировочные множители $N_{m,n}^1$  в формуле (10)
определяются путем предельного перехода при интегрировании
произведения функций Лежандра и первого из уравнений (7) [8, 15]
\begin{eqnarray}
\begin{array}{l}
    \displaystyle
N_{m,n}^1=\int\limits_{0}^{\alpha_0}
\,(P^{m}_{-\frac{1}{2}+i\,\nu_{m,n}}(\cosh\alpha))^2\,
\sinh\alpha\, d\alpha= \int\limits_{1}^{\cosh\alpha_0}
\,\left(P^{m}_{-\frac{1}{2}+i\,\nu_{m,n}}(z)\right)^2\, \, dz=
\\[5mm]
\displaystyle =\lim\limits_{\nu\to
\nu_{m,n}}\sinh^2\alpha_0\dfrac{P^{m}_{-\frac{1}{2}+i\,\nu}(z)-P^{m}_{-\frac{1}{2}+i\,\nu_{m,n}}(z)}{(1/4+\nu^2)-(1/4+\nu_{m,n}^2)}
\,\dfrac{dP^{m}_{-\frac{1}{2}+i\,\nu_{m,n}}(z)}{dz}\Bigg|_{z=\cosh\alpha_0}=
\\[5mm]
=\displaystyle \dfrac{\sinh^2\alpha_0}{2\nu_{m,n}}
P^{m\,\prime}_{-\frac{1}{2}+i\,\nu_{m,n}}(\cosh\alpha_0)
\dfrac{\partial
P^{m}_{-\frac{1}{2}+i\,\nu}(\cosh\alpha_0)}{\partial
\nu}\Bigg|_{\nu=\nu_{m,n}}.
\end{array}
\end{eqnarray}%11
%{\footnotetext{Вестник СПбГУ. Прикладная математика. Информатика... 2020. Т. 16. Вып. 3}}

Для вычисления неизвестных коэффициентов $A_m(\tau)$ и $D_{m,k}$ в
разложениях (3)--(5) используется метод перекрытия областей. В
силу того, что область {\bf 3}
частично\linebreak\pagebreak\noindentперекрывает области {\bf 1} и
{\bf 2},  граничные точки областей {\bf 1} и {\bf 2} при
$\beta=\beta_0$  совпадают (см. формулу (8)) и одновременно
являются внутренними точками  области {\bf 3} при $0\le
\alpha<\alpha_0$. Аналогично, граничные точки области {\bf 3} при
$\alpha=\alpha_0$ ($0\le \beta \le \beta_1$) представляют собой
внутренние точки  областей {\bf 1} ($\beta_0< \beta < \beta_1$) и
{\bf 2} ($0\le \beta < \beta_0$). Таким образом, непрерывность
потенциала и его первой производной во всей области диодной
системы, с учетом (2), обеспечивается выполнением условий
%
\begin{eqnarray}
U_1(\alpha,\beta_0,\phi)=U_2(\alpha,\beta_0,\phi)=\left\{
\begin{array}{lll}
U_3(\alpha,\beta_0,\phi), & 0\le \alpha<\alpha_0,  & -\pi\le \phi
< \pi,
\\[0.3mm]
f_0(\alpha,\phi), & \alpha_0\le \alpha<\infty,    & -\pi\le \phi <
\pi,
\end{array}
\right.
\end{eqnarray}%12
%
\begin{eqnarray}
U_3(\alpha_0,\beta,\phi)=\left\{
\begin{array}{lll}
U_2(\alpha_0,\beta,\phi), & 0\le \beta \le \beta_0, &  -\pi\le
\phi < \pi,
\\[0.3mm]
U_1(\alpha_0,\beta,\phi), & \beta_0 \le \beta \le \beta_1,  &
-\pi\le \phi < \pi.
\end{array}
\right.
\end{eqnarray}%13
%
%
%
%\vskip .5 cm
%
\noindent Из (2), (5), (8) и (12) для любого целого $m$ получим,
что
\begin{eqnarray*}
\int\limits_{0}^{\infty}B_m(\tau)
P^m_{-\frac{1}{2}+i\,\tau}(\cosh\alpha)\, d\tau=\;
\hspace*{-0.15cm} \left\{
\begin{array}{l}
    \hspace*{-0.15cm}
\sum\limits_{n=1}^{\infty}C_{m,n} \dfrac{\displaystyle
P^m_{-\frac{1}{2}+i\,\nu_{m,n}}(\cos\beta_0)}{\displaystyle
P^m_{-\frac{1}{2}+i\,\nu_{m,n}}(\cos\beta_1)}
P^m_{-\frac{1}{2}+i\,\nu_{m,n}}(\cosh\alpha)\,\,+
\\[4mm]  \displaystyle
\hspace*{-0.25cm} + \sum\limits_{k=1}^{\infty}D_{m,k}
\dfrac{\displaystyle P^m_{\mu_{m,k}}(\cos\beta_0)}{\displaystyle
P^m_{\mu_{m,k}}(\cosh\alpha_0)}   P^m_{\mu_{m,k}}(\cosh\alpha), \
0\le  \alpha<\alpha_0,
\\[3.5mm]
\hspace*{-0.15cm} \displaystyle \dfrac{\displaystyle
1}{\displaystyle 2\pi} \int\limits_{-\pi}^{\pi} e^{-im\phi}\,
f_0(\alpha,\phi)  \, d\phi,
 \displaystyle
\hspace*{0.2cm}
 \alpha_0\le \alpha<\infty,
\end{array}
\right.
\end{eqnarray*}%
%
%
%\vskip .5 cm
%
\noindent и, используя интегральное преобразование Мелера---Фока,
первое уравнение, свя\-зы\-ваю\-щее коэффициенты $B_m(\tau)$ и
$D_{m,k}$, можно представить в виде
\begin{eqnarray}
\begin{array}{l}
\displaystyle B_m(\tau)=(-1)^m \tau  \tanh(\pi \tau)  \times
\\[.5mm]
\displaystyle \times \Bigg[ \sum\limits_{k=1}^{\infty}D_{m,k}
\dfrac{\displaystyle P^m_{\mu_{m,k}}(\cos\beta_0)}{\displaystyle
P^m_{\mu_{m,k}}(\cosh\alpha_0)} \int\limits_{0}^{\alpha_0}
P^m_{\mu_{m,k}}(\cosh\alpha)
P^{-m}_{-\frac{1}{2}+i\,\tau}(\cosh\alpha) \sinh\alpha\,
d\alpha\,\,+
\\[4mm]
\hspace*{0.2cm}\displaystyle+ \sum\limits_{n=1}^{\infty}C_{m,n}
\dfrac{\displaystyle
P^m_{-\frac{1}{2}+i\,\nu_{m,n}}(\cos\beta_0)}{\displaystyle
P^m_{-\frac{1}{2}+i\,\nu_{m,n}}(\cos\beta_1)}

{\bf 5. Заключение.} В данной статье моделируется диодная
эмиссионная система с~полевым острием, поверхность которого
представляет собой гиперболоид вращения. Поверхностью анода может
быть либо часть гиперболоида вращения, либо круговая диафрагма. На
электродах заданы граничные условия первого рода, не обладающие
свойством осевой симметрии. Для вычисления трехмерного
распределения электростатического потенциала решена граничная
задача для уравнения Лапласа~(1),~(2) в~вырожденных
эллипсоидальных координатах для вытянутого эллипсоида вращения.
При решении поставленной задачи использовался метод перекрытия
областей, являющийся обобщением метода разделения переменных.
В~результате распределение потенциала представлено в виде
разложений по присоединенным функциям Лежандра~(3)--(7). Часть
коэффициентов, входящих в~разложения, вычислены в явном виде
(9)--(11) из граничных условий~(2). Нахождение остальных
коэффициентов сведено к решению системы линейных алгебраических
уравнений (14), (18). Кроме того, приведены формулы для
определения напряженности  поля (19)--(27). Распределение
потенциала и напряженности поля найдено во всей области системы.
