{\bf 1. Введение.}
 В работе рассматриваются вложенные методы решения начальной
задачи
\begin{eqnarray}
&y'(x) = f(x,y(x)),~~ y(x_{0})=y_0
     \label{system_0}
\end{eqnarray}
для  обыкновенных дифференциальных уравнений.

  В настоящее время для
 решения такой задачи широко используются [1,\,2] явные вло\-жен\-ные
методы  Рунге\,---\,Кутты
 RK$p(q)m$, $~p\neq q,\,$ $~p,q\in N.$ Здесь $p$~--- порядок метода вычисления приближения к
решению на шаге, $q$~--- порядок метода <<оценщика погрешности>>
полученного приближения, а $m$~--- число стадий. Данные методы можно
разделить на два класса. Для первого при  $~p<q~$ оценивается
погрешность аппроксимации [3], а для второго при $p>q$~---
последние учтенные в приближенном решении члены  c $q+1$-го до
$p$-го. Наиболее известный с середины 50-х годов XX~в. метод
второго класса был предложен В.~А.~Егоровым (см.~[1]). Это
расчетная схема RK4(2)4 (<<правило $^1\!/_6$>>) четвертого порядка c
контрольным членом Егорова. Она поч\-ти 20 лет была основной в данном
классе при создании алгоритмов интегрирования с автоматическим
выбором шага.

Работы Бутчера [4,\,5] дали толчок развитию алгебраической теории
методов Рунге\,---\,Кутты высоких порядков и конструированию на их
основе вложенных методов двух клас\-сов. Так, для второго класса в
[6] были построены расчетные схемы пятого порядка двух типов
RK5(4)7M,\,RK5(4)7S, RK5(4)6M. Формально при равенстве точностных
ха\-ракте\-ри\-стик первые две расчетные схемы RK5(4)7M,\,RK5(4)7S
тре\-буют б\'ольших вычислительных затрат,\,чем RK5(4)6M. Используемая
при построении этих методов технология FSAL (First Same As Last~---
последнее вычисление правой части на текущем шаге является первым
на следующем)  устраняет фактически данный недостаток. Для методов
такого типа в аббре\-виа\-ту\-ру методов добавляется латин\-ская~<<F>>.

Вычислительная схема RK5(4)6 так же, как и расчетные схемы
RK6(5)8M, RK8(7)13M, представленные в [7], относится к стандартным
классическим методам этого класса.


В работе строятся семейства семистадийных методов шестого порядка,
вложенных методов шестого порядка RK6(4)7 второго класса
стандартного типа и вложенных методов шестого порядка RK6(4)8F.


Предполагаем достаточно гладкой правую часть рассматриваемого
дифферен\-циального уравнения. Считаем, что известно точное решение
$y(x_n)$ задачи \eqref{system_0} в точке $x_n\in [X_0,X_k]$.

Приближение $p$-го порядка --- $z_{n+1}$ и $q$-го --- $\hat
z_{n+1}$ к точному решению $y(x_{n+1})$ в~точках $x_{n+1}\in
[X_0,X_k]$, $x_{n+1}=x_{n}+h_{n},$ $h_n=\theta (x_n)h$ и $0<\theta
(x_{n})\leq 1,$ $n=0,1,2,\ldots,$ $z_{n}=y(x_n)$ ищем
соответственно в виде
\begin{eqnarray}
&&z_{n+1} = z_{n}+\sum_{\nu=1}^{m}b_{\nu}k_{\nu}(h), \quad \Vert y(x_{n+1})-z_{n+1}\Vert\approx O(h^{p+1}),\label{nulgr11}\\
&&\hat z_{n+1} = z_{n}+\sum_{\nu=1}^{m}\hat b_{\nu}
k_{\nu}(h),\quad \Vert y(x_{n+1})-\hat z_{n+1}\Vert\approx
O(h^{q+1}),\label{nulgr12}
\end{eqnarray}
вычисляя $k_{\nu}\equiv k_{\nu}(h)$  по формулам
\begin{eqnarray}
    &&k_{\nu}=h_{n}f\biggl(x_{n}+c_{\nu}\,h_{n},z_{n}+\displaystyle\sum_{\mu=1}^{\nu-1}a_{\nu\mu}k_{\mu}\biggr),\label{nulgr13}
    \end{eqnarray}
где  $b_{\nu}, c_{\nu}, a_{\nu,\mu}$ ---  параметры метода; $h_n $~--- шаг интегрирования.

Разность двух приближений ($E_{n+1}= \Vert z_{n+1}-\hat
z_{n+1}\Vert - $ контрольный член) используется [1,\,2] для
управления величиной шага $h_n$ интегрирования:
\begin{eqnarray}
h_{n+1}=0.9h_{n}\left(\frac{\hbox{\rm
tol}}{E_{n+1}}\right)^{\frac{1}{q+1}}, \label{nulgr14}
\end{eqnarray}
где (5) tol~--- максимально допустимое значение контрольного члена.

\par {\bf 5. Сравнительное тестирование.} {\it Орбита Аренсторфа.}
Плоское движение кос\-ми\-чес\-кого аппарата (КА) с координатами
$(x_1,x_2)$ в гравитационном поле, создаваемом Землей $(0,0)$ и
Луной $(1,0),$ описывается [2] системой обыкновенных
дифференциальных уравнений
\begin{equation}
        \begin{aligned}
        \ddot{x}_1&=x_1 + 2\dot{x}_2 - \mu'\frac{x_1+\mu}{D_1} - \mu\frac{x_1-\mu'}{D_2},\\
        \ddot{x}_2&=x_2 - 2\dot{x}_1 - \mu'\frac{x_2}{D_1} -
        \mu\frac{x_2}{D_2},\label{testz1}
    \end{aligned}
    \end{equation}
где~ %
%\[
%   \quad \
$ D_1 = ((x_1+\mu)^2 + x_2^2)^{3/2};~~ D_2 = ((x_1-\mu')^2 + x_2^2)^{3/2};~~ $%
%\]
%
%\[
   $ \mu = 0.012277471;~~ \mu' = 1-\mu.\linebreak $%
%\text{%
При~начальных условиях%} \]
%
\begin{align}
    x(0) &= (0.994,\, 0),\quad
    \dot{x} (0) = (0,\, 2.00158510637908252240537862224)\label{testz2}
\end{align}
КА движется по орбите с периодом $T_{{\rm per}} =
17.0652165601579625588917206249$.




В сравнительном тестировании наряду с представленными
здесь вложенными методами
(RKS6(4)7$\Bigl[\frac{5}{21}\Bigr]$\,(см.\,табл.\,2),
 RKS6(4)7$\Bigl[ -\frac{625}{96}\Bigr]$\,(см.\,табл.\,3),
 RKS6(4)8F\,(табл.\,5)) шестого
по\-ряд\-ка в качестве оппонентов при решении задачи
\eqref{testz1},\,\eqref{testz2}  использовали
 методы того же класса (RK5(4)7M $\equiv$ DOPRI5(4)7F,\,RK5(4)7S [4],\,RK6(5)8M [5]).

\par Вычисления выполнялись с числами с плавающей запятой и мантиссой длиной 30. Максимально допустимое значение контрольного члена ${\rm tol}$ (см. (5)) изменялось в
диапазоне ${\rm tol}\in [10^{-24},10^{-4}]$ для всех расчетных
схем (RKS6(4)7$\Bigl[\frac{5}{21}\Bigr]$ (см.\,табл.\,2),
 RKS6(4)7$\Bigl[ -\frac{625}{96}\Bigr]$ (см.\,табл.\,3),
 RKS6(4)8F (табл.\,5), RK5(4)7M~$\equiv$ DOPRI5(4)7F, RK5(4)7S~[4], RK6(5)8M [5]) с единым алгоритмом \eqref{nulgr14} вычисления шага интег\-ри\-ро\-вания.

 На рис.~1,\,{\it а,\,б} и рис.~2 приведены
 зависимости полной погрешности ${\rm Err_{glob}}$ от  общего количества вычислений правой части $N_f$ на интервале $[0,T_{\rm per}]$.
