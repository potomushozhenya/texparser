 {\bf Введение.}
В работе [1] дан алгоритм приведения систем обыкновенных
диф\-фе\-рен\-циаль\-ных уравнений
\begin{eqnarray*}
    z'_k = \varphi_k(x,z_1,\ldots,z_g),\quad k=1,\ldots,g,
\end{eqnarray*}
к виду
\begin{eqnarray}
&&y'_{0} = f_{0}(x,y_{0},\ldots,y_{n}),
    \label{system_0} \\
&&y'_{i} = f_{i}(x,y_{0},\ldots,y_{i-1},y_{l+1},\ldots,y_{n}),
    \qquad i=1,\ldots,l, \label{system_i} \\
&&y'_{j} =   f_{j}(x,y_{0},\ldots,y_{j-1}),
    \qquad j=l+1,\ldots,n, \label{system_j}
\end{eqnarray}
где

\longrightarrow \mathbb{R}^{r_{s}} , & \quad s=0,\ldots,n, \end{equation*}%\\
\begin{equation*}f_{0}:[X_{0},X_{k}] \times \mathbb{R}^{g}\ \ \longrightarrow

\noindent Группы уравнений \eqref{system_i}, \eqref{system_j},
называемые {\itshape структурно выделенными,} структурно
тождественны. Каждое уравнение одной из этих групп  уравнений
занимает определенное место в~последовательности уравнений своей
группы. Его правая часть не зависит от искомых функций, поведение
которых описывается этим и~всеми последующими уравнениями той же
группы. Группа уравнений \eqref{system_0}, в~которую вошли все
уравнения, не имеющие структурных особенностей указанного выше
типа, будем называть {\itshape общей}. Она, как и~группа уравнений
\eqref{system_i}, может отсутствовать. Необходимость
в~интегрировании систем такого типа возникает, например, в~задачах
небесной механики и~физики высоких энергий.

В работах [2--4] предложено обобщение явного метода Рунге---Кутты
для интегрирования структурно разделенных систем
\eqref{system_0}--\eqref{system_j}.
 В~основе рассматриваемого обобщения лежит алгоритмическое
использование выделенных структурных особенностей. Эффективность
построенных
 методов определяется тем, что для общей группы уравнений
\eqref{system_0} численное интегрирование по соотношению порядка
метода ($q$) и~минимально возможного числа этапов ($m$)
тождественно методам Рунге---Кутты ($m>q$, $q\geqslant 5$), а~для
структурно выделенных групп уравнений \eqref{system_i},
\eqref{system_j} предложенная модификация дает возможность
уменьшить число этапов при сохранении порядка метода. Причем
в~случае отсутствия общей группы уравнений такое уменьшение еще
более значительно. Так, метод пятого порядка получен не за шесть
этапов, а~за четыре [5], метод шестого порядка [6--10]~--- за
шесть этапов вместо семи.

В настоящей работе для полной (содержащей общую группу уравнений)
системы \eqref{system_0}--\eqref{system_j} ({\it полная
каноническая форма}) строится экономичный метод шестого порядка:
семистадийный по общей группе и~шестистадийный по структурно
выделенным.

\par{\bf Метод интегрирования.}
Считаем, что нам известно точное решение $y_{s}(x)$,
$s=0,1,\ldots,n,$ системы \eqref{system_0}--\eqref{system_j}
в~то\-ч\-ке $x\in [X_0,X_k]$. Не умаляя общности рассуждений, для
простоты вывода примем, что $r_s=1$, $s=0,1,\ldots,n$.

Для численного интегрирования систем
\eqref{system_0}--\eqref{system_j} рассмотрим явный одношаговый
метод типа Рунге---Кутты, который в~дальнейшем будем называть
{\itshape структурным} и~обозначать RKS. В~предположении
достаточной гладкости правой части изучаемой системы приближение
$z_{s}$ к~точному решению $y_{s}(x+h)$, $s=0,1,\ldots,n,$ в~точке
$x+h\in [X_0,X_k]$ ищем в~виде
\begin{equation}
y_{0}(x+h) \approx z_0 = y_{0}(x)+\sum_{\nu=1}^{m_{0}}b_{0,\nu}
k_{0,\nu}(h),   \label{nulgrup}
\end{equation}
\begin{equation}
y_{i}(x+h) \approx z_i = y_{i}(x)+\sum_{\nu=1}^{m_{1}}b_{1,\nu}
k_{i,\nu}(h), \quad i=1,\ldots,l,  \label{igrup}
\end{equation}
\begin{equation}
y_{j}(x+h) \approx z_j = y_{j}(x)+\sum_{\nu=1}^{m_{2}}b_{2,\nu}
k_{j,\nu}(h), \quad j=l+1,\ldots,n,  \label{jgrup}
\end{equation}
причем $k_{s,w}\equiv k_{s,w}(h)$ вычисляются в~строгой
последовательности:
\begin{equation}
    k_{0,1},\ k_{1,1},\ \ldots, \ k_{n,1},\ k_{0,2},\ k_{1,2},\ \ldots,\
    k_{n,2},\ k_{0,3},\ k_{1,3},\ \ldots \label{solk}
\end{equation}
по формулам
\begin{eqnarray*}
    &k_{0,\nu}=f_{0}\biggl(x+c_{0,\nu}\,h,y_{0}(x)+\displaystyle\sum_{\mu=1}^{\nu-1}a_{00,0,\mu}k_{0,\mu},&\\
       &\qquad y_{1}(x)+\displaystyle\sum_{\mu=1}^{\nu-1}a_{01,1,\mu}k_{1,\mu},\ldots,y_{l}(x)+\displaystyle\sum_{\mu=1}^{\nu-1}a_{01,l,\mu}k_{l,\mu},&\\
&\qquad
y_{l+1}(x)+\displaystyle\sum_{\mu=1}^{\nu-1}a_{02,l+1,\mu}k_{l+1,\mu},\ldots,y_{n}(x)+\displaystyle\sum_{\mu=1}^{\nu-1}a_{02,n,\mu}k_{n,\mu}\biggr),&\\
&k_{i,\nu}=f_{i}\biggl(x+c_{1,\nu}\,h,y_{0}(x)+\displaystyle\sum_{\mu=1}^{\nu}a_{10,0,\mu}k_{0,\mu},&\\
   &\qquad y_{1}(x)+\displaystyle\sum_{\mu=1}^{\nu}a_{11,1,\mu}k_{1,\mu},\ldots,y_{i-1}(x)+\displaystyle\sum_{\mu=1}^{\nu}a_{11,i-1,\mu}k_{i-1,\mu},&\\
&\qquad
y_{l+1}(x)+\displaystyle\sum_{\mu=1}^{\nu-1}a_{12,l+1,\mu}k_{l+1,\mu},\ldots,y_{n}(x)+\displaystyle\sum_{\mu=1}^{\nu-1}a_{12,n,\mu}k_{n,\mu}\biggr),\quad i=1,2,\ldots,l,&\\
&k_{j,\nu}=f_{j}\biggl(x+c_{2,\nu}\,h,y_{0}(x)+\displaystyle\sum_{\mu=1}^{\nu}a_{20,0,\mu}k_{0,\mu},&\\
   &\qquad y_{1}(x)+\displaystyle\sum_{\mu=1}^{\nu}a_{21,1,\mu}k_{1,\mu},\ldots,y_{l}(x)+\displaystyle\sum_{\mu=1}^{\nu}a_{21,l,\mu}k_{l,\mu},&\\
&~~~~
y_{l+1}(x)+\displaystyle\sum_{\mu=1}^{\nu}a_{22,l+1,\mu}k_{l+1,\mu},\ldots,y_{j-1}(x)+\displaystyle\sum_{\mu=1}^{\nu}a_{22,j-1,\mu}k_{j-1,\mu}\biggr),\quad
j=l+1,\ldots,n,&
    \end{eqnarray*}
где  $b_{u,\nu}, c_{u,\nu}, a_{uv,\nu,\mu} $  --- параметры
метода; $h$~--- шаг интегрирования.

{\bf Замечание.} В~соответствии с~требованием
 алгоритма \eqref{solk} для компонентов вектора числа этапов (стадий)
$M=(m_{0},m_{1},m_{2})$ структурного метода
\eqref{nulgrup}--\eqref{jgrup} характерными являются соотношения
\begin{equation*}
m_{0} \geqslant m_{1}\geqslant m_{2}.
\end{equation*}

Выбор параметров метода осуществляется таким образом, чтобы для
метода шестого порядка разложение погрешности метода
\eqref{nulgrup}--\eqref{jgrup} на шаге по степеням $h$
\begin{equation*}
    \vert y_{s}(x+h) - z_{s}\vert\approx O(h^{7}),\qquad
    s=0,1,\ldots,n,
\end{equation*}
начиналось с~седьмой степени  при выбранном числе этапов
$M=(m_0,m_1,m_2).$

\par{\bf Условия порядка.}
В рамках метода  \eqref{nulgrup}--\eqref{jgrup} шестого порядка
как число условий порядка, связывающих параметры метода, так
и~количество самих параметров метода зависит от количества
выделенных групп уравнений в~исходной системе
\eqref{system_0}--\eqref{system_j}.

Известно, что условия порядка для семистадийного метода
($m_{0}=7$) Рунге---Кут\-ты шестого порядка образуют систему 37
нелинейных уравнений с~35 неизвестными параметрами: $C_{0},$
$B_{0},$ $A_{00}.$

 При построении шестиэтапных ($m_{1}=6,\, m_{2}=6$) методов шестого порядка [6--9] для систем
\eqref{system_0}--\eqref{system_j}, не содержащих общей
 группы  уравнений \eqref{system_0},  условия
порядка образуют систему 74 нелинейных алгебраических уравнений
с~61 неизвестным параметром $C_{1},C_{2},$ $B_{1},B_{2},$
$A_{12},A_{21}$ при  $l=1,\,n=2.$ Если же  $l>1,\, n-l>1,$ то
число нелинейных уравнений в~условиях порядка возрастает до 292,
а~неизвестных параметров $C_{1},C_{2},$ $B_{1},B_{2},$
$A_{11},A_{12},A_{21},A_{22}$~--- до 103.

\par {\bf Тестирование.}
%\par Для численного тестирования RKS$[766](1/4,\,7/9)$ построена система
%дифференциальных уравнений
Для проведения численного тестирования
 на решении
$y_{0}(x)=\exp{\left(4\sin{x^{2}}\right)},\,$
$y_{1}(x)=\exp{\left(5\sin{x^{2}}\right)},\,$
$y_{2}(x)=\exp{\left(\sin{x^{2}}\right)},\,$ $
 y_{3}(x)=\cos{\left(x^{2}\right)},\,$
 $y_{4}(x)=\sin{\left(x^{2}\right)}+1,$
 все компоненты которого удовлетворяют начальным условиям
$y_{s}(0)=1,\,s=0,1,\ldots,4,$ построена система дифференциальных
уравнений
\begin{equation}
    \begin{array}{@{\,}l@{\,}@{\,}c@{\,}@{\,}l@{\,}@{\,}c@{\,}@{\,}l@{\,}}
y_{0}^{'}&=&\,x\,y_{3}\bigl(\,y_{1}\,y_{2}^{-1}+7\,y_{0}\bigr)&=&f_{0}(x,y_{0},y_{1},y_{2},y_{3}),\\
y_{1}^{'}&=&\,10x\,\exp(5(y_{4}-1))y_{3}&=&f_{1}(x,y_{3},y_{4}),\\
y_{2}^{'}&=&\,\displaystyle{2x\,y_{1}^{{1\over 5}}\,y_{3}+\frac{1}{4}\ln{y_{0}}-y_{4}+1}&=&f_{2}(x,y_{0},y_{1},y_{3},y_{4}),\\
y_{3}^{'}&=&\,-\displaystyle{\frac{2}{5}x\,\ln{(y_{0}\,y_{2})}}&=&f_{3}(x,y_{0},y_{2}),  \\
y_{4}^{'}&=&\,2x\,y_{0}\,y_{1}^{-1}\,y_{2}\,y_{3}&=&f_{4}(x,y_{0},y_{1},y_{2},y_{3}), \\
y_{i}(0)  &=&\,1,\, x\in[0,T_k],\,i=0,1,\ldots,4,&&\,
F=(f_{0},f_{1},\ldots,f_{4})^{T}.
\end{array}\label{test_2}
 \end{equation}

 Тестовая задача \eqref{test_2} содержит все три группы
уравнений полной канонической формы системы
\eqref{system_0}--\eqref{system_j}. Общая группа уравнений состоит
из первого уравнения системы \eqref{test_2}. Две пары последующих
уравнений этой системы попадают соответственно в~первую и~вторую
структурно выделенные группы уравнений.

Применение метода RKS$6[7,6,6](1/4,\,7/9)$ для интегрирования
системы \eqref{test_2} позволит уменьшить число обращений
к~процедуре вычисления правой части на единицу для
$f_{1},\ldots,f_{4}$ с~сохранением порядка метода. Причем в~этом
случае проверку правильности найденных значений параметров метода
проходят параметры всех девяти блоков $A_{uv}.$

При отказе от алгоритмического использования структурных
особенностей системы \eqref{test_2} (все пять уравнений образуют
общую группу \eqref{system_0} канонической формы) метод
RKS$6[7,6,6](\alpha,\,\beta)$ вырождается в~классический способ
распространения метода Рунге---Кутты шестого порядка на системы.
Он образует однопараметрическое семейство RKS$6[7](\beta)$
семиэтапных методов относительно параметра $\beta.$ Для
определенности в~дальнейшем будем обозначать RKS$6[7](7/9)$ его
расчетную схему при $\beta=7/9.$



В~качестве оппонента методам RKS$6[7,6,6](1/4,\,7/9)$
и~RKS$6[7](7/9)$ была использована
 семиэтапная
 расчетная схема RKB$6$ шестого порядка Бутчера [11].

 Наклоны ломаных на рисунке показывают, что для
всех тестируемых методов зависимость глобальной погрешности
$Err_{glob}$ от величины шага интегрирования (числа обращений
$N_{f}$ к~процедурам вычисления правых частей) имеет шестой
порядок. Порядок методов RKS$6[7](7/9)$ и~RKS$6[7,6,6](1/4,\,7/9)$
действительно соот\-ветст\-вует заявленному шестому. При этом
и~RKS$6[7](7/9),$ и~метод-оппонент требуют семи вычислений правой
части СОДУ на шаге, тогда как RKS$6[7,6,6](1/4,\,7/9)$~--- только
шести для компонент решения принадлежащих второй \eqref{system_i}

 подтвердило порядок методов RKS$6[7,6,6](1/4,\,7/9)$
и~RKS$6[7](7/9)$  и~их работоспособность, а~также показало при

методы численного интегрирования систем обыкновенных
дифференциальных уравнений.
