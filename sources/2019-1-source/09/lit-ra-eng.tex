{\leftskip=7mm\noindentВ статье исследована конфликтно-управляемая
модель динамики мнений в социальной сети с двумя центрами влияния.
Центры посредством выбора уровней влияния на остальных участников
сети преследуют определенные интересы. При рассмотрении этой
модели как некооперативной игры двух лиц и выборе в качестве ее
решения равновесия по Нэшу найдены равновесные уровни влияния для
каждого из центров.\\[1mm]
\textit{Ключевые слова}: социальная сеть, влияние, динамика
мнений, равновесие.
